## 3. 기존 발행 글 향상 제안

> 첫 보고서로 기존 글 목록 미확인. 향후 발행 글에 즉시 적용할 표준 체크리스트.

### 내부 링크 구성 가이드
- 모든 신규 글에 **최소 2개의 내부 링크** 삽입
  - 예: "돌침대 쿠션 선택법" 글 → "흙침대 자세 교정" 글 연결
  - 예: "허리 통증 원인" 글 → 제품 리뷰 글 연결
- 링크 앵커 텍스트는 키워드 포함 자연어 사용 (예: "돌침대 전용 등받이 쿠션 리뷰 보기")

### 이미지 ALT 텍스트 수정 기준
| 잘못된 예 | 올바른 예 |
|-----------|----------|
| `image001.jpg` | `돌침대-등받이-쿠션-허리지지-사용예시` |
| `제품사진` | `따숨마루-흙침대-쿠션-측면각도` |
| `img_003` | `허리통증-완화-등받이쿠션-40대-추천` |

- ALT 텍스트는 **15~30자** 이내, 핵심 키워드 + 상황 묘사 조합

### 소제목(H2/H3) 개선 원칙
- 의문형 소제목 적극 활용 → 네이버 VIEW 알고리즘 체류 시간 증가
  - 예: "등받이 쿠션이란?" → "왜 돌침대에는 일반 쿠션이 안 될까요?"
  - 예: "제품 특징" → "따숨마루 쿠션이 허리를 편안하게 하는 3가지 이유"
- 소제목에 **숫자 포함** 시 클릭률(CTR) 향상 효과 있음

## 4. 오늘의 블로그 SEO 전략 인사이트

### 타이틀 패턴 분석 (네이버 상위 노출 패턴)

**고성과 타이틀 구조:**
```
[타겟 상황] + [핵심 키워드] + [해결 또는 추천]
```
- 예: `돌침대에서 허리 아프신 분` + `등받이 쿠션` + `이렇게 고르세요`
- 예: `50대 허리 통증` + `흙침대 쿠션` + `전문가 추천 3선`

**피해야 할 패턴:**
- 광고성 느낌 강한 타이틀: "OO 쿠션 강력 추천!", "무조건 이거!"
- 금칙어 포함 타이틀 (치료, 완치, 업계 1위 등)
- 키워드 나열형: "쿠션 등받이 허리 돌침대 흙침대"

### 연관글 구성 전략

**클러스터 구조 (허브 앤 스포크 모델):**
```
[허브 글] 돌침대·흙침대 등받이 쿠션 완전 가이드
    ├─ [스포크 1] 돌침대 쿠션 선택법
    ├─ [스포크 2] 흙침대 앉는 자세 교정
    ├─ [스포크 3] 허리 통증 원인과 생활 개선
    ├─ [스포크 4] 따숨마루 제품 상세 리뷰
    └─ [스포크 5] 구매 후기 모음
```
- 허브 글을 먼저 발행하고 스포크 글들이 허브로 내부 링크 → 허브 글 도메인 파워 집중
- 스포크 글 간 상호 연결로 체류 시간·페이지뷰 증가

### 네이버 VIEW 알고리즘 최적화 포인트
1. **발행 주기:** 주 3회 이상 꾸준히 (화·목·토 권장)
2. **글 길이:** 최소 1,200자, 이상적으로 2,000~2,500자
3. **이미지 수:** 글당 3~5장 (첫 이미지가 가장 중요)
4. **댓글·공감 유도:** 글 말미 질문형 마무리로 댓글 유도
5. **발행 시간:** 오전 7~9시 또는 오후 8~10시 (타겟 연령대 활동 시간)

## 5. 네이버 상위 노출 콘텐츠 질 개선 체크리스트

### 발행 전 필수 체크 (Pre-publish)
- [ ] 타이틀에 메인 키워드 포함 (앞쪽 배치)
- [ ] 타이틀 25~35자 이내
- [ ] 도입부 첫 2문장 안에 메인 키워드 자연 포함
- [ ] 소제목(H2) 3개 이상, 소제목에 키워드 포함
- [ ] 본문 1,200자 이상
- [ ] 이미지 3장 이상, 모든 이미지 ALT 텍스트 작성
- [ ] 내부 링크 2개 이상
- [ ] 스마트스토어 링크 1개 (자연스러운 문맥 삽입)
- [ ] 금칙어 미포함 확인 (치료·완치·의학적효과·업계1위·최고·무조건·100%·절대·완벽)
- [ ] 글 말미 독자 참여 유도 문구

### 발행 후 관리 체크 (Post-publish)
- [ ] 발행 후 24시간 내 네이버 검색 노출 여부 확인
- [ ] 1주 후 조회수·공감수 기록
- [ ] 유사 경쟁 글 대비 노출 순위 점검
- [ ] 댓글 24시간 내 응답
- [ ] 월 1회 본문 내용 업데이트 (날짜 갱신 효과)

### 콘텐츠 톤앤매너 가이드
- **전문적이지만 친근하게:** 40~60대 독자 눈높이 (쉬운 단어, 짧은 문장)
- **공감 우선:** 통증·불편함 먼저 공감 → 정보 제공 → 제품 자연 연결
- **과장 금지:** 경험담·사용 후기 기반의 신뢰감 있는 서술
- **시각적 구성:** 소제목·줄바꿈·이모지(최소화) 활용으로 가독성 확보
