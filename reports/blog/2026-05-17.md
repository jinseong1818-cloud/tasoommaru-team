- 오전 10시~오후 2시: 느린 탐색·감성 콘텐츠 소비 피크 (홈리빙, 건강)
- "감성형" "일상형" 콘텐츠 체류 시간 주중 대비 +35~40% 상승
- 주말 동안 쌓인 구매 욕구 → 일요일 오후 최종 결정 패턴

**오늘 최적 타이틀 패턴**:
```
[느린 일상 장면] + [내가 달라진 경험] — 감성형 서술 구조
```

**권장 타이틀 예시 (일요일 감성 + 구매 확신 지원)**:
- `돌침대 위에서 가장 편안한 순간 — Cozy-라인이 만들어준 오후` ← 오늘 발행 적용
- `등받이쿠션 하나 바꿨더니 달라진 것 — 돌침대 사용 2개월 후기`
- `일요일 오후, 나만의 공간을 만들어준 흙침대 쿠션 이야기`

**피해야 할 패턴**:
- ~~`Cozy-라인 완벽 소개`~~ (금칙어 + 정보 나열형)

- ~~`허리에 무조건 좋은 쿠션`~~ (금칙어 + 의학적 주장)
