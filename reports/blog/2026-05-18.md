### 주제 A — 추천 1순위 (메인 허브 글)

**제목**: `돌침대·흙침대 등받이쿠션 고르는 법 — A-라인 vs Cozy-라인, 내 몸에 맞는 선택 기준`

**타겟**: 돌침대·흙침대 사용자 중 등받이쿠션 구매 결정 단계의 독자 (비교·선택 탐색자)

**도입부 방향 가이드**:
```
✅ 오프닝 (구매 결정 공감 훅, 120자):
"등받이쿠션 하나 사려고 찾아보면 종류가 너무 많죠.
소파용인지, 침대용인지도 헷갈리고 — 돌침대에 맞는 건지 아닌지는
사고 나서야 아는 경우가 많아요.
오늘은 돌침대·흙침대 사용자 기준으로만 얘기해볼게요."

✅ 브랜드 신뢰 확보 (금칙어 없이):
"따숨마루는 돌침대·흙침대 전용 설계로
특허·디자인 등록을 마친 브랜드예요.
누적 판매 300건, 리뷰 150개 이상 쌓이는 동안
가장 많이 받은 질문이 있어요. '두 라인 중 뭘 사야 해요?'"

✅ 콘텐츠 구조 (선택 기준 체크리스트 + 비교표):
① 내 사용 패턴 체크: 눕기 전 잠깐 vs 30분 이상 앉아 있음
② 내 체형 체크: 등 곡선이 뚜렷한 편 vs 평평한 편
③ 내 공간 체크: 쿠션이 잘 보이는 위치 vs 감추는 편
④ A-라인 vs Cozy-라인 스펙 비교표 (구조·소재·크기·추천 대상)
⑤ 실제 리뷰어 유형별 선택 패턴 ("이런 분은 A-라인", "이런 분은 Cozy-라인")

✅ 내부 링크 배치 (필수 3곳):
→ A-라인 상세 후기 글 (5/15): "A-라인 자세한 후기는 여기서 →"
→ Cozy-라인 상세 후기 글 (5/17): "Cozy-라인 감성 후기는 여기서 →"
→ 브릿지 글 (5/18): "돌침대에서 등받이가 필요한 순간이 궁금하다면 →"

✅ CTA (자연 전환):
"어떤 라인이 맞는지 결정하셨다면, 스마트스토어에서 바로 확인해보세요.
A-라인·Cozy-라인 둘 다 같은 페이지에서 고르실 수 있어요. → [링크]"
```

**A-라인 특성 반영 포인트**:
- 삼각형 웨지 구조·완만한 경사 → "허리 곡선을 살리면서 경사를 만드는 구조"
- 전면 패널 수평 봉제선 → "단정하고 정돈된 외관 — 인테리어 노출 공간에 적합"
- 추천 사용 패턴: 잠깐 기대거나 허리 곡선이 뚜렷한 분

**Cozy-라인 특성 반영 포인트**:
- 풍성한 쿠션감·라운드 형태 → "감싸는 느낌, 소파처럼 안기는 구조"
- 극세사 스웨이드 소재 동일 → "두 라인 모두 같은 촉감, 차이는 형태와 지지 방식"
- 추천 사용 패턴: 30분 이상 앉아 TV·독서하는 분

**필수 포함 키워드**: 돌침대등받이쿠션, 흙침대쿠션, 등받이쿠션추천, 허리건강쿠션, 돌침대쿠션선택, 따숨마루, A라인, Cozy라인  
**권장 글자 수**: 2,000~2,500자 (정보 허브 — 체류 시간 극대화 목표)  
**이미지 구성**:
1. A-라인·Cozy-라인 나란히 비교 (대표 썸네일 — 두 라인 한눈에)
2. A-라인 측면 웨지 구조 강조 컷
3. Cozy-라인 라운드 형태 부감 컷
4. 50대 여성 모델 — A-라인 사용 자세 (각도 강조)
5. 동일 모델 — Cozy-라인 사용 자세 (감싸는 느낌)
6. 스마트스토어 링크 삽입용 두 라인 단독 제품컷

## 5. 타겟 고객 공감 콘텐츠 방향 제안

> 타겟: 한국 여성, 50대 초중반, 건강·홈리빙 관심층 / 월요일 = 주말 탐색 완료 후 구매 결정 최종 전환 시점

### 5-1. 월요일 타겟 감정·상황 맵핑

| 월요일 생활 장면 | 감정 | 콘텐츠 접근 방향 |
|--------------|------|---------------|
| 주말 내내 찾아봤는데 아직 결정 못 함 | 결정 피로·확신 필요 | "이것만 보고 결정하세요" — 단순화된 선택 기준 제공 |
| 출근 전 잠깐 스마트폰으로 탐색 | 시간 압박 + 빠른 결론 욕구 | 체크리스트·비교표 포맷 — 짧게 스캔 가능한 구조 |
| 퇴근 후 돌침대에 기대며 오늘도 불편했음 | 오늘도 불편했다는 현실 자각 | "오늘 퇴근하고 돌침대에 앉았을 때 또 불편하셨다면" |
| 자녀한테 받은 선물로 주문해볼까 고민 | 자기 배려·선물 수용 욕구 | "나를 위한 작은 변화 — 허리 편한 일상이 그것" |
| 리뷰 더 읽어볼까, 그냥 주문할까 망설임 | 사회적 증거 탐색 | "150명이 먼저 써본 이유" — 리뷰 수 자연 언급 |

### 5-2. 월요일 최적 공감 콘텐츠 포맷 (우선순위 순)

**① 선택 기준 제시형** (내일 허브 글 핵심 포맷)
- "A-라인이 맞는 분 / Cozy-라인이 맞는 분" 체크리스트
- 독자가 읽으면서 "나는 Cozy-라인이네" 자연스럽게 결론 도달
- 선택 완료 → CTA 클릭 전환 최적화

**② 문제 공감 + 솔루션형** (오늘 브릿지 글 포맷)
- "돌침대 쓰면서 이런 불편함 있으셨죠?" — 고개를 끄덕이게 만드는 오프닝
- 문제 3가지 공감 → 솔루션 1가지 자연 전환
- 구매 결정 직전 독자를 마지막으로 잡아주는 역할

**③ 사회적 증거 중심형** (5/22 비교 글 적용 권장)
- "300명 구매자가 A-라인보다 Cozy-라인을 더 많이 선택한 이유"
- "리뷰 150개에서 공통으로 나온 표현 3가지"
- 수치·리뷰 중심 → 결정 피로 독자에게 가장 강력한 신호

### 5-3. 비주얼 방향 — 월요일 구매 결정자 설득 포인트

```
오늘 브릿지 글 + 내일 허브 글 이미지 공통 방향:
① 두 라인 나란히 비교 컷 — 크기·형태 차이 한눈에
② 사용 자세 비교 (A-라인: 허리 각도 강조 / Cozy-라인: 감싸는 느낌 강조)
③ 극세사 스웨이드 손 클로즈업 — 소재 공통점 시각화 (두 라인 같은 소재)
④ 50대 여성 모델 — 리넨 와이드팬츠 + 니트 카디건 / 자연광 / 오크 바닥
⑤ 차콜 그레이 + 웜 베이지 색상 양쪽 노출 (색상 선택 고민 독자 배려)

썸네일 (허브 글):
→ "A-라인 | Cozy-라인" 텍스트 오버레이 + 두 제품 나란히
→ 크림·베이지 기조 배경 / 따숨마루 로고 우하단
```
