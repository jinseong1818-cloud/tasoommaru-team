### 주제 A — 추천 1순위 (시즌 건강 정보형 스포크)

**제목**: `에어컨 켜기 시작하면 허리가 더 피곤해지는 이유 — 실내 생활 자세 이야기`

**도입부 방향 가이드**:
```
✅ 오프닝 (시즌 공감 훅, 130자):
"5월 말이 되면서 슬슬 에어컨을 켜기 시작했는데,
왜인지 허리가 더 뻐근한 느낌이 드는 분 계신가요?
저도 처음엔 그냥 날씨 탓인가 했는데 — 생활 자세 때문이었더라고요."

✅ 시즌 연결 + 생활 문제 제기:
"에어컨이 켜지면 창문을 닫고 실내에서 보내는 시간이 길어지죠.
바깥 활동은 줄고, 앉아있는 시간은 늘고 — 그 사이 허리는 조금씩 쌓이는 피로를 감당하고 있어요.
특히 돌침대나 흙침대 위에서 오래 앉아계신 분들은 여름이 생각보다 허리에 부담이 될 수 있어요."

✅ 콘텐츠 구조:
① 에어컨 시즌 실내 생활 증가 → 앉는 시간 증가 → 허리 피로 누적 흐름 (의학 표현 없이 생활 언어로)
② 돌침대·흙침대 위에서 오래 앉을 때 자세 이야기 (생활 관찰형, 진단·처방 표현 금지)
③ 실내 허리 관리 루틴 3가지 (가벼운 스트레칭·자세 교정 쿠션·휴식 루틴 — 생활 조언형)
④ 등받이 쿠션이 여름 실내 생활에 어떻게 도움이 되는지 (제품 직접 언급은 후반부 1섹션)
⑤ A-라인·Cozy-라인 여름 사용 팁 간략 소개 + 스마트스토어 자연 연결

✅ 내부 링크 배치 (필수 3곳):
→ 허브 글(5/19): "A-라인과 Cozy-라인 중 어떤 게 맞는지 고민이라면 →"
→ 2기 스타터 글(5/26): "A-라인 3주 실사용 후기에서 여름 사용 이야기도 다뤘어요 →"
→ Cozy-라인 감성 글(5/28): "Cozy-라인 감촉 이야기는 여기서 →"

✅ CTA:
"실내 생활이 길어지는 요즘, 등받이 쿠션 하나가 생각보다 큰 차이를 만들더라고요.
A-라인·Cozy-라인 모두 스마트스토어에서 보실 수 있어요. → [링크]"
```

**제품·브랜드 반영 포인트**:
- 돌침대·흙침대 **전용 설계** 차별점: "일반 쿠션과 달리 돌침대 특유의 단단한 표면을 고려한 설계" 자연 삽입
- 극세사 스웨이드 소재: "여름에도 덥지 않은 소재" 언급 가능 (소재 실제 특성 기반)
- 색상: 차콜 그레이(시원한 톤) / 웜 베이지(자연스러운 톤) 여름 인테리어 연결 가능

**필수 포함 키워드**: 50대 허리 건강, 돌침대등받이쿠션, 흙침대쿠션, 등받이쿠션추천, 허리쿠션, 따숨마루  
**금칙어 최종 점검**: `치료` / `완치` / `의학적 효과` / `업계 1위` / `최고` / `무조건` / `100%` / `절대` / `완벽`  
**권장 글자 수**: 1,800~2,200자 (건강 정보형 — 신뢰도 있는 정보 제공 + 생활 공감 균형)  
**이미지 구성**:
1. 에어컨 켜진 실내 돌침대 위 앉아있는 50대 여성 — 자연스러운 생활 장면 (자연광 + 실내 조명 혼합)
2. 등받이 없이 앉았을 때 vs 등받이 쿠션 있을 때 자세 비교 (직관적 비주얼)
3. A-라인 또는 Cozy-라인 제품 실사용 장면 — 여름 실내 분위기
4. 스트레칭·자세 루틴 간략 인포그래픽 (이미지 ALT에 키워드 반영)
5. 스마트스토어 연결 CTA 배너 (마지막 이미지)

## 5. 타겟 고객 공감 콘텐츠 방향 제안

> 타겟: 한국 여성, 50대 초중반, 건강·홈리빙 관심층  
> **금요일 심리 맵**: 한 주 마감 안도감 + 주말 기대감 + 자기 배려 욕구 최고점. "나를 위해 뭔가 하나 챙겨도 되겠다"는 허용감이 가장 높은 요일.

### 5-1. 금요일 타겟 감정·상황 맵핑

| 금요일 생활 장면 | 감정 | 콘텐츠 접근 방향 |
|--------------|------|---------------|
| 퇴근 후 돌침대에 기대서 드디어 쉬는 순간 | 안도감 + 피로 해소 욕구 | "금요일 저녁, 드디어 돌침대에 기댈 수 있는 시간 — 이 자리가 조금 더 편해지면 좋겠다는 생각이 들더라고요" |
| 이번 주말 제대로 쉬고 싶다는 생각 | 주말 기대감 + 자기 배려 | "이번 주말은 특별한 것 없이 집에서 쉬기로 했어요. 그럼 그 시간이 편해야 하지 않나요?" |
| 리뷰 150개 쌓인 걸 보며 믿음이 생기는 순간 | 구매 직전 최종 확신 | "누적 판매 300건, 리뷰 150개 넘었을 때 — 이게 갑자기 신뢰가 가더라고요" |
| 에어컨 켜면서 몸이 더 뻐근해진 경험 | 시즌 불편함 공감 | "에어컨 틀면서 오히려 허리가 더 피곤해진 것 같다는 느낌, 저만 그런 게 아니죠?" |
| 비싸지 않은데 효과가 있을까 고민 | 가성비 기대 + 불안 | "큰 돈 들이지 않아도, 자세 하나 잡는 것만으로 달라질 수 있다는 게 — 이게 써보면 알게 되더라고요" |

### 5-2. 주말 준비 독자를 위한 콘텐츠 포맷

**① 자기 배려 허용 프레임** (금요일 퇴근 후 독자 최적)
- "한 주 수고했으니까, 이번 주말만큼은 나를 조금 더 챙겨도 괜찮아요"
- 제품이 "선물" 또는 "자기 배려"의 수단으로 포지셔닝 — 스스로에게 주는 작은 선물
- 가격 언급 시: "부담 없는 금액으로, 생활 자세 하나가 달라지는 경험" 방식으로 자연 삽입

**② 사회적 증거 활용** (구매 직전 최종 확신 탐색 독자)
- "누적 판매 300건 이상, 리뷰 150개 이상 — 비슷한 분들이 이미 경험해보셨어요"
- 특허·디자인 등록: "돌침대·흙침대 전용으로 설계된 제품이라 일반 쿠션과는 구조가 달라요" — 1줄로 간결하게
- 리뷰에서 자주 나오는 표현 2~3가지 자연 인용 ("처음 기댔을 때 놀랐다", "소재가 생각보다 고급스럽다")

**③ 에어컨 시즌 시의성 콘텐츠** (내일 5/30 글 핵심 포맷)
- "5월 말이 되면서 달라진 생활 패턴" 으로 시즌 유입
- 에어컨 = 실내 생활 증가 = 앉는 시간 증가 = 자세·등받이 관심 증가 → 제품 연결 고리 자연 형성
- 건강 정보는 생활 관찰 언어로만: "피곤함", "뻐근함", "자세가 무너지는 느낌" (의학 표현 금지)

```
금요일 저녁·주말 최적 비주얼 방향:
① 따뜻한 저녁 실내 조명 + 오후 자연광 혼합 — 아늑하고 쉬고 싶은 공간 분위기

⑤ 원형 러그 + 오크 바닥 + 자연광 창문 미니멀 배경 유지
```
