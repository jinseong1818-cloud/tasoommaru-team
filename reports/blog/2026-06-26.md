### 오늘의 키워드 전략 포인트

- **`흙침대쿠션` D+1 진입 확인이 이번 주 가장 중요한 단일 체크포인트**: 경쟁이 거의 없는 키워드이고 스마트스토어팀이 동일 키워드를 상품명에 반영하기로 결정했으므로, D+1 시점에 진입 신호가 보이면 양 채널 동시 강화 효과가 실제로 작동하는지 첫 검증이 가능. 미진입이면 어제 글의 키워드 배치를 재점검할 시점
- **금요일은 "한 주 콘텐츠 정리 + 주말 준비"형 검색이 늘어나는 요일**: 거창한 신규 정보 탐색보다 "이번 주 본 글들을 다시 찾아본다"는 재방문성 검색 비중이 높아지므로, 오늘 브릿지의 내부 링크 구성(돌침대·흙침대 2부작 회고)이 노출량보다 체류시간·재방문에 더 크게 기여할 가능성
- **`여름허리쿠션`은 D+5로 초기 성장 구간의 마지막 관찰 시점에 근접**: 다음 점검(D+6~7)에서 안정화 전환 여부가 판가름되므로, 오늘 중 순위 추이를 기록해두면 전환 시점 판단의 기준값이 됨

## 4. 오늘의 블로그 SEO 전략 인사이트

### 4-1. 금요일 + 콘텐츠 정리·주말 전환일 특화 SEO 전략

**금요일 건강·홈리빙 카테고리 검색 행동 패턴**:
- 오전(8~11시): 한 주 마무리를 의식하며 검색 의도가 "정리·재방문"형으로 전환 — 이번 주 본 콘텐츠를 다시 찾는 비중 증가
- 오후(13~17시): 주말 계획을 세우며 생활 전반 탐색성 검색이 증가, 날씨(장마)에 따른 실내 활동 검색 동반 상승
- 저녁(19~22시): 금요일 저녁은 한 주 피로 해소와 주말 기대가 겹치는 시간 — 다만 오늘은 신규 정보 탐색보다 "이번 주 읽은 글의 후속"을 찾는 검색이 더 유효

**오늘 발행 글 최적 타이틀 구조**:
```
[생활 장면(주말·장마)] + [시간의 길이감 표현] + [제품 자연 등장] — 브릿지·정리 톤

오늘 발행: "장마철 주말, 집에 머무는 시간이 길어질 때"

금요일 브릿지 타이틀 패턴:
✅ "[계절·요일] 주말, ~한 시간이 길어질 때" (생활 장면 중심, 정리 톤)
✅ "비 오는 금요일 ~" (날씨·요일 장면 중심)
❌ "장마철 허리 통증 무조건 해결법" (금칙어 "무조건" 포함 — 절대 사용 금지)
```

### 4-2. 클러스터 흐름 관리 — "장마철 2부작 마무리" → "주말 라인 선택" 진입 구간

```

동선 A (이번 주 2부작 유입자 → 회고·정리): 6/24 돌침대 스포크 ↔ 6/25 흙침대 스포크 → 오늘 브릿지

→ 오늘로 "장마철 2부작"이 콘텐츠 측면에서 한 번 정리되었으므로, 내일부터는 주말 동안 A-라인(6/20)·Cozy-라인(6/27)을 균형 있게 다루는 흐름으로 전환하는 것이 자연스러움. 한쪽 라인에 콘텐츠가 편중되지 않도록 라인별 노출 균형을 주간 단위로 점검할 시점.
```
