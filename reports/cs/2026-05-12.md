## 2. 문의 유형별 모범 답변 스크립트

### 2-1. 배송 일정 및 현황 문의

**답변 스크립트:**
```
안녕하세요, 따숨마루입니다 😊
소중한 문의 감사드립니다.

주문 번호 [OOO]번 확인했습니다.
현재 [배송사명] 기준 운송장 번호 [OOO]로 배송 중이며,
예상 도착일은 [날짜]입니다.

실시간 배송 조회는 아래 링크에서 확인 가능합니다.
[배송 조회 링크]

추가 문의 사항은 언제든지 말씀해 주세요!

**주의사항:**
- 배송 지연 시 사전 안내 문자 발송 필수
- 지연 사유(물량 증가, 기상 등) 구체적으로 안내

### 2-2. 제품 사이즈·규격 확인

> **고객:** "돌침대에 맞는 사이즈인지 확인하고 싶어요."

**답변 스크립트:**
```
안녕하세요, 따숨마루입니다 😊

따숨마루 등받이 쿠션의 규격은 다음과 같습니다.

• 높이: 약 55cm (삼각형 구조 경사면 기준)
• 너비: 약 60cm
• 두께(하단 최대): 약 30cm

돌침대·흙침대의 폭이 90cm 이상이라면 안정적으로 사용 가능하며,
침대 헤드 없이 벽면에 기대어 사용하실 때도 편리합니다.

고객님의 침대 규격을 알려주시면 더 정확하게 안내해 드리겠습니다!
```

### 2-3. 교환·반품·환불 요청

빠르게 처리해 드리겠습니다.

항상 더 나은 제품으로 보답하겠습니다. 감사합니다.

**답변 스크립트 (금칙어 주의):**
```
안녕하세요, 따숨마루입니다 😊

따숨마루 등받이 쿠션은 돌침대·흙침대 위에서
허리를 편안하게 지지해주는 것을 목적으로 설계된 제품입니다.

삼각형 인체공학 구조로 허리 곡선에 자연스럽게 밀착되어
장시간 앉아 계실 때 자세 유지에 도움을 드릴 수 있습니다.

다만, 허리 관련 증상의 진단·치료는 반드시 의료 전문가와 상담하시길 권장합니다.
편안한 자세 지지용 제품으로 사용 시 많은 분들이 만족해 주시고 있습니다 😊

더 궁금하신 점 있으시면 편하게 물어보세요!

**⚠️ 절대 금지 표현:** "치료", "완치", "디스크에 효과", "의학적으로 증명" 등

## 3. FAQ 업데이트 제안 (신규 Q&A 2개)

### Q1. 돌침대 말고 일반 매트리스에도 사용할 수 있나요?
**A.**  
따숨마루 등받이 쿠션은 주로 돌침대·흙침대처럼 단단한 바닥면을 가진 침대에 최적화되어 있습니다. 일반 스프링 매트리스처럼 푹신한 침대에서는 쿠션이 뒤로 밀려날 수 있어 안정적인 지지가 어려울 수 있습니다. 안정적인 하드 타입 침대 또는 바닥 좌식 환경에서 가장 효과적으로 사용하실 수 있습니다.

---

### Q2. 제품에 특허나 디자인 등록이 되어 있나요?
**A.**  
네, 따숨마루 등받이 쿠션은 디자인 등록 및 특허 등록이 완료된 제품입니다. 독자적인 삼각형 구조와 인체공학적 설계를 법적으로 보호받고 있으며, 유사 모방 제품과 품질 면에서 차별화된 정품임을 보증합니다.

## 4. 고객 불만 패턴 분석 및 예방 전략

### 주요 불만 패턴

| 패턴 | 원인 | 빈도 |
|------|------|------|
| "생각보다 딱딱해요" | 온라인 사진과 실물 질감 차이 | 높음 |
| "배송이 너무 늦어요" | 성수기 물량 집중, 산간 지역 | 중간 |
| "사이즈가 맞지 않아요" | 구매 전 규격 미확인 | 중간 |
| "허리 효과가 없어요" | 사용 방법 미숙지, 기대 과다 | 낮음 |

### 예방 전략

**1. 상품 상세페이지 강화**
- 실제 사용 환경(돌침대 위) 사진 및 영상 추가
- 규격 비교 인포그래픽 (A4용지, 일반 베개 대비)
- "이런 분께 추천합니다 / 맞지 않는 경우" 섹션 명시

**2. 구매 후 자동 안내 메시지 발송**
- 배송 출발 시: 운송장 번호 + 예상 도착일
- 수령 2일 후: 올바른 사용법 가이드 링크 발송
- 수령 7일 후: 만족도 확인 + 리뷰 요청

**3. 상품 설명에 기대치 조율 문구 삽입**
- "허리 통증 완화를 위한 의학적 제품이 아니며, 편안한 자세 지지용으로 제작되었습니다"
