### 최종 선정: **"밥 먹고 어딜 기대요?"**

**선정 이유:**
- 식후 자세 고민자(세그먼트 D)의 일상 순간을 한 문장에 압축 — 밥을 먹고 나서 기댈 곳을 찾지 못하는 경험은 30~50대 누구나 공감
- 질문형 구조로 "나 얘기인가?" 반응 유발 → 스크롤 멈춤 효과 극대화
- 10자로 짧고 리듬감이 있어 대형 텍스트 오버레이로도 시각적 임팩트 유지
- "기대다"라는 단어가 제품 행위(등받이 쿠션에 기대기)와 자연스럽게 연결

## STEP 2. 4컷 씬 구성표

### [씬 1] 후킹 씬 (0~3초)

| 항목 | 내용 |
|------|------|
| **목적** | 손가락을 멈추게 함 |
| **상황 설정** | 식사 후 소파/바닥에 어색하게 앉아 허리를 구기거나, 벽에 비스듬히 기대다 불편해하는 30~40대 인물 |
| **핵심 장면** | 인물이 자세를 여러 번 바꾸다 결국 반쯤 누운 어정쩡한 자세로 폰을 보는 클로즈업 |
| **텍스트 오버레이** | "밥 먹고 어딜 기대요?" (대형 Bold 고딕, 화면 중앙) |
| **권장 카메라** | Dutch angle — 약간 기울어진 앵글로 긴장감·문제 상황 연출, 허리 곡선이 두드러지게 |

---

### [씬 2] 공감 씬 (3~8초)

| 항목 | 내용 |
|------|------|
| **목적** | "맞아, 나도 이래" 공감 유발 |
| **상황 설정** | 딱딱한 소파 등받이 or 맨벽에 기댄 측면 샷 — 허리가 C자로 굽고, 인물이 불편한 표정으로 자세를 고쳐 앉는 반복 행동 |
| **핵심 장면** | 측면에서 척추 곡선이 명확히 드러나는 자세 → 고쳐 앉아도 계속 무너지는 반복 동작 |
| **텍스트 오버레이** | "이 자세, 나만 반복하나요?" (얇은 명조, 하단 1/3) |
| **권장 카메라** | Side profile medium shot — 척추 커브가 선명하게 보이는 순수 측면 미디엄샷, 고정 카메라 |

---

### [씬 3] 솔루션 씬 (8~13초)

| 항목 | 내용 |
|------|------|
| **목적** | 제품이 문제를 해결하는 순간 |
| **상황 설정** | 따숨마루를 소파 등받이 / 벽에 밀착 세팅 → 인물이 자연스럽게 기대며 허리가 펴지는 장면 |
| **핵심 장면** | ① 제품 삼각 웨지 구조 클로즈업 (차콜 그레이 스웨이드 질감) → ② 인물이 기대는 순간 S자 자세로 안착 |
| **텍스트 오버레이** | "3중 레이어, 꺼지지 않는 지지력" (Bold 고딕, 화면 상단) |
| **권장 카메라** | Rack focus — 배경 흐림 상태에서 제품으로 포커스 이동 → Medium shot으로 전환하며 인물 자세 변화 포착 |

---

### [씬 4] 클로징 씬 (13~15초)

| 항목 | 내용 |
|------|------|
| **목적** | 감성 여운 + 행동 유도 |
| **상황 설정** | 식후 여유로운 시간 — 따숨마루에 기댄 채 따뜻한 음료를 들고 평온하게 쉬는 30~40대 인물의 일상 컷 |
| **핵심 장면** | 따뜻한 햇살이 드는 거실, 인물이 자연스럽게 기대며 미소 짓는 와이드샷 |
| **텍스트 오버레이** | "기대도 됩니다, 바르게 ✦ 링크는 프로필에 🔗" (브랜드 서체, 중앙 하단) |
| **권장 카메라** | Slow pull back — 서서히 멀어지며 따뜻한 거실 공간 전체가 보이는 감성적 엔딩 |
