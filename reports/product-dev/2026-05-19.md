## 2. 기존 제품 개선 아이디어

### A-라인 개선안 (2가지)

**① 분리형 탈착 커버 시스템 도입**
- 현재: 일체형 극세사 스웨이드 커버
- 개선: YKK 지퍼 방식의 탈착 커버 + 내부 메모리폼 본체 분리 구조
- 고객 니즈: 세탁 편의성 (리뷰 불만 2위 직접 해소)
- 추가 효과: 계절별 커버 교체 판매 → 객단가 상승 (커버 단품 판매 가능)
- 난이도: 중 / 예상 추가 원가: +3,000~5,000원
- 우선순위: ★★★★★

**② 후면 논슬립 실리콘 패턴 패드 적용**
- 현재: 후면 패브릭 마감 (미끄러짐 발생)
- 개선: 후면 하단부 실리콘 도트 패턴 또는 TPU 논슬립 패널 부착
- 고객 니즈: 돌침대·황토침대 표면에서의 안정적 고정 (리뷰 불만 1위 직접 해소)
- 추가 효과: '전용 설계' 핵심 메시지 더욱 강화 → 상세페이지 어필 포인트 신규 추가 가능
- 난이도: 하 / 예상 추가 원가: +800~1,500원
- 우선순위: ★★★★★

### Cozy-라인 개선안 (2가지)

**① 내부 충전재 레이어 이중 구조 도입**
- 현재: 단일 충전재 (라운드 형태의 풍성한 쿠션감)
- 개선: 하단 저반발 메모리폼(지지) + 상단 마이크로비즈 or 폴리에스터(포근함) 이중 레이어
- 고객 니즈: 장시간 사용 시 꺼짐 방지 + 첫 감촉의 포근함 유지
- 추가 효과: "오래 앉아도 꺼지지 않는다"는 신규 USP 확보
- 난이도: 중 / 예상 추가 원가: +4,000~7,000원
- 우선순위: ★★★★

**② 암레스트(팔걸이) 일체형 Cozy Plus 라인 분기**
- 현재: 등받이 단독 구조
- 개선: Cozy 라인에 좌우 암레스트 패드를 일체화한 U자형 확장 디자인
- 고객 니즈: "TV 볼 때 팔 둘 곳이 없다" 잠재 불만 선제 해소
- 추가 효과: 고관여 시니어 선물 수요 공략 (설/추석 선물세트 포지셔닝 가능)
- 난이도: 높음 / 예상 추가 원가: +12,000~18,000원 → 별도 SKU로 분기
- 우선순위: ★★★ (중장기 검토)

## 3. 파생·변형 제품 아이디어

### 🆕 따숨마루 좌식 방바닥 등받이쿠션 — "마루라인"

**컨셉**: 돌침대·황토침대에서 벗어나 **한국 좌식 생활 전반**으로 타겟 확장  
**배경**: 50대 타겟 고객은 돌침대 외에도 방바닥·마루·좌식 소파에서 TV 시청·식사를 하는 경우가 많음. 현재 A/Cozy 라인은 침대 매트리스 높이에 최적화 → 방바닥에 세우면 각도·높이가 맞지 않음  

**차별점**:
- 방바닥(낮은 높이)에 최적화된 낮은 웨지각(15°) 재설계
- 후면에 벽 밀착용 플랫 패널 + 바닥 논슬립 테두리 일체형
- 좌식 방석과 세트 구성 판매 가능 (마루 등받이 + 마루 방석 패키지)

**타겟**: 현 고객층 + 허리 통증 있는 좌식 생활자 (타겟 연령대 동일, 구매 맥락 확장)  
**예상 가격**: 38,000~45,000원 (기존 라인과 유사 포지셔닝)  
**우선순위**: ★★★★ (단기 기획 검토 가능)

- 밀도: 45~50D (등받이용 적정 범위)
