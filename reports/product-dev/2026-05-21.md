## 2. 기존 제품 개선 아이디어

### A-라인 개선

**"논슬립 더블 고정 시스템" 적용**
- 현황: 단단한 돌침대 표면 위에서 쿠션이 미끄러지는 불만이 반복 접수됨
- 개선안: 하단면 전체에 실리콘 논슬립 패턴 가공 + 허리 부위를 감싸는 탄성 밴드(고무줄 허리 벨트) 추가
- 기대 효과: "돌침대 전용"이라는 USP를 기능적으로 완성, 경쟁 제품과의 명확한 차별화
- 추가 생산 원가 예상: 개당 약 800~1,200원 증가 (실리콘 코팅 + 밴드)

### Cozy-라인 개선

**"사계절 듀얼 커버" 구조 도입**
- 현황: 풍성한 쿠션감이 강점이나 여름철 더위 불만이 A-라인 대비 더 높음 (두꺼운 소재)
- 개선안: 본체(심재)와 커버를 분리 구조로 전환 → 기본 커버(극세사 스웨이드) + 여름용 쿨링 메쉬 커버 1장 동봉 세트
- 기대 효과: 연중 사용 가능 → LTV 증가, 커버 단품 재구매 유도 채널 생성
- 가격 전략: 기본 세트 + 여름 커버 동봉으로 가격대 프리미엄 포지셔닝 유지

## 3. 파생·변형 제품 아이디어

### "따숨마루 힐링 슬라이드 보조매트" (돌침대 전용 좌식 매트)

- **개요:** 돌침대 위에 깔아 쿠션과 함께 사용하는 얇은 논슬립 좌식 패드 (두께 1~2cm, 폴리우레탄 폼 + 극세사 커버)
- **기획 근거:**
  - 고객 불만 1위 "미끄러짐"을 쿠션 본체 개조 없이 별도 제품으로 해결 → 세트 판매 가능
  - 돌침대 사용자는 침대 위에 오래 앉으므로 엉덩이·허벅지 받침 니즈 존재
  - 따숨마루 브랜드 색상(차콜 그레이 / 웜 베이지)으로 동일 출시 → 세트 구매 유도
- **예상 가격대:** 18,000~25,000원 (단품) / 등받이쿠션 세트 번들 5,000원 할인
- **시너지:** 리뷰 고객 업셀링, 등받이쿠션 단독 구매 대비 객단가 30~40% 향상 기대

## 5. 소싱 추천 제품

### 추천: 인체공학 쿨링 메쉬 등받이 쿠션 커버 원단 (국내 위탁 OEM)

- **소재:** 3D 에어메쉬 + 탄성 밴드 구조 (여름용 커버 전용)
- **소싱처 유형:** 국내 섬유 OEM 업체 (경기·충남 산업단지 소재 다수) / 리빙바이원B2B 계열 도매 플랫폼 활용 가능
- **활용 방안:**
  - Cozy-라인 "사계절 듀얼 커버" 개선안의 여름 커버 소재로 즉시 적용
  - 최소 발주 수량(MOQ) 100~200장 수준으로 파일럿 테스트 가능
- **예상 단가:** 커버 한 장 소싱 원가 3,000~6,000원 (사이즈·마감 수준에 따라)
- **우선 액션:** 국내 OEM 업체 2~3곳 샘플 요청 → 기존 A/Cozy 쿠션 본체에 피팅 테스트

## 6. 시너지 제품 아이디어

### 아이디어 1: "돌침대 전용 무릎 받침 쿠션" (반원형 볼스터)

- **근거:** 돌침대에 등받이 쿠션을 사용할 때, 다리를 뻗거나 반가부좌 자세를 취하면 무릎이 딱딱한 침대 표면에 닿아 불편 → 무릎 받침 쿠션 니즈 자연 발생
- **형태:** 반원형 볼스터(원통형), 지름 약 15cm, 극세사 스웨이드 동일 소재, 차콜 그레이 / 웜 베이지
- **판매 방식:** 단품 판매 + 등받이 쿠션 2종 세트 번들 기획
- **예상 가격:** 단품 22,000~28,000원 / 세트 할인 10% 적용

### 아이디어 2: "허리 온열 패치 호환 파우치 포켓" (기능 업그레이드 키트)

- **근거:** 돌침대 사용 고객은 허리 통증이 주 구매 동기 → 온열 패드·핫팩 병행 사용 가능성 높음
- **아이디어:** 등받이쿠션 뒷면 또는 하단에 포켓 구조 추가 → 시중 판매 온열 패치·미니 핫팩 삽입 공간 확보
- **구현 방법:** 기존 A-라인 신규 버전에 지퍼 포켓(약 20×15cm) 봉제 추가
- **차별점:** 온열 기능을 직접 내장하지 않아 KC 인증 부담 없이 온열 연계 경험 제공 가능
