## 2. 기존 제품 개선 아이디어

### A-라인 개선

**"멀티 포지션 웨지 인서트" 높이 선택 구조 도입**

- **현황:** 삼각형 웨지 구조는 경사가 고정되어 있어 키가 크거나 작은 고객, 또는 침대 높낮이가 다른 고객에게 핏이 맞지 않는다는 피드백 증가
- **개선안:** 쿠션 내부에 두께가 다른 보조 웨지 인서트(1cm 증감 단위, 2~3개 레이어) 삽입·제거 구조 도입 → 고객이 자신의 체형에 맞춰 높이를 직접 조절 가능
- **기대 효과:** "나한테 안 맞는다"는 리뷰 불만 해소, 맞춤형 포지셔닝으로 프리미엄 SKU 신설 가능
- **추가 원가 예상:** 인서트 레이어 2개 추가 시 약 1,500~2,500원 증가 / 소비자가 3,000~5,000원 인상으로 마진율 유지 가능

### Cozy-라인 개선

**"리무버블 넥롤 세트" 번들 구성**

- **현황:** 풍성한 쿠션감이 강점이나 허리만 받쳐주는 구조로, 오래 앉을 때 목·어깨 피로 호소가 증가하고 있음
- **개선안:** Cozy-라인 상단에 탈부착 가능한 넥롤(원통형 목 받침, 지름 약 12cm) 추가 → 벨크로 또는 버튼 방식으로 본체에 연결·분리 가능
- **기대 효과:** 허리 + 목 동시 케어 → "전신 상체 지지 쿠션"으로 포지셔닝 격상, 단가 프리미엄 정당화
- **가격 전략:** 넥롤 포함 Cozy-Pro 버전으로 신규 SKU 등록, 기존 Cozy-라인은 엔트리 유지

## 3. 파생·변형 제품 아이디어

### "따숨마루 S·L 사이즈 에디션" — 체형별 맞춤 라인

- **개요:** 현재 단일 사이즈(약 50cm×45cm)로 출시 중인 A-라인·Cozy-라인을 S(소형, 43cm×38cm)와 L(대형, 58cm×52cm) 2가지 추가 사이즈로 확장
- **기획 근거:**
  - 오늘 리뷰 인사이트에서 "높이·크기가 내 체형에 안 맞음" 불만이 불만 1위로 집계됨
  - 돌침대 규격 자체가 일반 침대보다 낮아 사용자 체형 편차가 체감 핏에 크게 영향을 미침
  - S 사이즈: 키 155cm 이하 소형 체형 또는 어린이·청소년 사용자, 선물 구매 (조부모님께 대형 + 어머니께 소형 등 패밀리 번들 기회)
  - L 사이즈: 키 170cm 이상 또는 체구가 크고 넓은 등받이를 원하는 고객
- **출시 전략:** 단일 색상(웜 베이지만) 시범 출시 후 반응 확인 → 차콜 그레이 확대
- **예상 가격대:** S — 39,000원 / L — 52,000원 (기존 M 포지셔닝 대비 각각 -10% / +15%)
- **시너지:** 리뷰 내 사이즈 불만 해소, 가족 단위 세트 번들 기획 연계 가능

## 5. 소싱 추천 제품

### 추천: 항균·방습 메모리폼 인서트 레이어 (국내 폼 OEM)

- **소재:** 저반발 메모리폼(30D 이상) + 항균 처리 원단 래핑, 두께 1cm / 2cm 단일 규격
- **소싱처 유형:** 국내 폼 전문 제조 OEM (경기 안성·충남 천안 소재 폼 공장 다수 운영 중) / 알리바바 국내 에이전트 통한 초도 샘플 확보도 가능
- **활용 방안:**
  - A-라인 "멀티 포지션 웨지 인서트" 개선안의 레이어 소재로 즉시 적용 가능
  - 메모리폼 특성상 체중에 맞춰 압력 분산 → 기존 폼보다 체감 높이 조절 효과 향상
  - 항균 기능 추가 시 "위생 관리" 소구점 한 개 더 확보 (돌침대 장기 사용자의 위생 민감도 높음)
- **예상 단가:** 1cm 레이어 1개당 원가 800~1,500원 / MOQ 500개 이상 시 단가 하락 가능
- **우선 액션:** 국내 폼 OEM 업체 2곳 샘플 요청 (두께 1cm / 2cm 각각) → A-라인 실물 핏 테스트 후 채택 여부 결정

## 6. 시너지 제품 아이디어

### 아이디어 1: "따숨마루 넥롤 쿠션" (목·어깨 전용)

- **근거:** 오늘 불만 TOP2 "목 받침이 없음" 리뷰에서 등받이쿠션 사용 시 목·어깨 추가 지지 니즈 확인. Cozy-라인 개선안의 탈부착 넥롤을 단독 제품으로도 출시하여 기존 고객의 업셀링 채널 형성
- **형태:** 반원통형 넥롤, 길이 약 40cm · 지름 12cm, 극세사 스웨이드 동일 소재, 동일 색상 2종
- **판매 방식:** ① 단품 판매 ② 등받이쿠션 + 넥롤 세트 번들 (10% 할인) ③ Cozy-Pro 버전에 기본 동봉
- **예상 가격:** 단품 24,000~30,000원

### 아이디어 2: "따숨마루 선물세트 패키지" (기프팅 라인)

- **근거:** 오늘 칭찬 TOP1 "선물용으로 딱"과 트렌드 분석의 "시니어 기프팅" 성장이 교차하는 포인트. 현재는 별도 선물 포장 옵션 없이 기본 배송 박스만 존재
- **구성안:** 등받이쿠션(A 또는 Cozy) + 넥롤 쿠션 + 브랜드 쇼핑백 + 손글씨 스타일 카드(감사·건강 메시지 인쇄) 세트
- **포지셔닝:** "부모님께 드리는 건강 선물 No.1", 어버이날(5월)·추석·설날 시즌 집중 프로모션
- **예상 가격:** 65,000~75,000원 (번들 구성 대비 10% 할인 적용)
- **마케팅 접점:** 스마트스토어 "기획전" 탭 등록 + 인스타그램·유튜브 선물 언박싱 콘텐츠 연계
