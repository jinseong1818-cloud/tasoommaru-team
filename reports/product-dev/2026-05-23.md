## 2. 기존 제품 개선 아이디어

### A-라인 개선

**후면 논슬립 패드 통합 — "고정형 A-라인" 업그레이드**

- **현황:** A-라인은 심플한 삼각형 웨지 구조로 심미성은 높으나, 돌침대 벽면과의 밀착 고정 기능이 없어 사용 중 위치 이탈 문제가 꾸준히 보고됨
- **개선안:** 쿠션 후면 하단 1/3 영역에 얇은 실리콘 논슬립 패드(3mm 두께, 도트 패턴) 봉제 삽입 → 벽 표면과의 마찰력 증대로 사용 중 밀림 현상 방지
- **추가 소구 포인트:** "고정 유지 구조" 텍스트를 상세페이지 기능 배지(뱃지)로 추가 → 경쟁 제품과의 기능 차별화 강화
- **원가 영향:** 실리콘 패드 소재 원가 약 300~600원 / 봉제 공정 추가 약 200원 → 총 500~800원 원가 증가, 소비자가 유지 가능
- **구현 난이도:** 낮음 (기존 봉제라인 변경 없이 후면 추가 부착 공정만 추가)

### Cozy-라인 개선

**썸머 쿨링 커버 교체형 세트 — "Cozy 시즌 커버 시스템" 도입**

- **현황:** Cozy-라인의 풍성한 쿠션감은 겨울·봄 시즌에 강점이나, 5~9월 온열 불편 호소로 여름 시즌 구매·재구매가 둔화되는 패턴 예상
- **개선안:** 기존 극세사 스웨이드 커버와 교체 가능한 여름용 쿨링 커버(냉감 아이스실크 또는 메쉬 원단, 동일 색상 2종) 별도 판매 → 본체는 그대로 두고 커버만 시즌별 교체
- **판매 방식:** ① 쿨링 커버 단품 (14,000~18,000원) ② Cozy 본체 + 쿨링 커버 세트 (기존가 + 10,000원) ③ 연간 구독형 "시즌 커버 정기배송" 장기 검토
- **기대 효과:** 여름 시즌 매출 방어 + 기존 구매 고객의 업셀링 채널 신설
- **우선 액션:** 냉감 아이스실크 원단 샘플 3종 조달 → Cozy 본체에 핏 테스트

## 3. 파생·변형 제품 아이디어

### "따숨마루 플로어 쿠션 (좌방석형)" — 바닥 생활 확장 라인

- **개요:** 돌침대·흙침대 사용자 중 다수가 전통적 한국식 바닥 생활(방바닥·거실 바닥에 앉기)을 병행한다는 점에 착안, 등받이쿠션과 페어링되는 방석형 좌방석 쿠션 개발
- **기획 근거:**
  - 타겟 고객(50대 한국 여성)은 한식 주방·거실 바닥 생활 비율이 타 연령대 대비 높음
  - 기존 등받이쿠션이 침대 사용 외 거실 바닥·명상 공간·온돌방 사용 사례도 리뷰에서 확인됨
  - 방석 시장은 이미 경쟁이 많지만, "따숨마루 특유의 극세사 스웨이드 + 인체공학 설계"를 적용한 고급 좌방석은 차별화 포지셔닝 가능
- **제품 스펙 (안):** 사각형 또는 라운드, 45cm×45cm, 두께 8~10cm 고탄성 폼, 극세사 스웨이드 동일 소재·동일 컬러 적용 → 등받이쿠션과 세트 비주얼 완성
- **출시 전략:** 등받이쿠션 구매 후 "이 제품과 함께 구매하면 좋은 상품" 스마트스토어 크로스셀링 연동, 세트 할인 구성
- **예상 가격:** 단품 28,000~35,000원 / 등받이쿠션 + 방석 세트 65,000~72,000원
- **리스크:** 방석 시장 경쟁 심화 → 독자 브랜드 포지셔닝 명확화 필요 (기성 방석과 소재·설계 면에서 명확한 차이 어필 필수)

## 5. 소싱 추천 제품

### 추천: 냉감 아이스실크 커버 원단 (여름 시즌 쿨링 커버용)

- **소재 규격:** Q-Max 0.4 이상 냉감지수 보유 아이스실크 원단, 혼방률 나일론 80% + 폴리에스터 20% 또는 비슷한 규격, 폭 150cm 이상
- **소싱처 유형:** 국내 원단 도매상(동대문 원단시장 도매처 또는 경기 파주 소재 원단 전문 업체) / 직수입 시 중국 광저우 원단 박람회(Canton Fair) 이후 에이전트 연결 가능
- **활용 방안:**
  - Cozy-라인 교체형 쿨링 커버 개발의 핵심 소재
  - 냉감 원단은 극세사 스웨이드 대비 얇고 견고 → 봉제 공정 단순화 기대
  - 동일 원단으로 A-라인 여름 커버 버전도 동시 개발 가능 → 소재 통합 구매로 단가 절감
- **예상 단가:** 원단 1m당 4,000~7,000원 (Q-Max 수치·두께에 따라 상이) / 쿠션 1개 커버 소요량 약 0.8m → 원단 원가 3,200~5,600원
- **우선 액션:** 샘플 원단 3종(냉감지수 상이한 3가지) 구매 → 기존 극세사 스웨이드와 나란히 놓고 촬영 후 내부 비주얼 심의 진행

## 6. 시너지 제품 아이디어

### 아이디어 1: "따숨마루 허리 온열 쿠션 커버" (USB 저온 발열 내장형)

- **근거:** 타겟 고객(50대 허리 통증)이 돌침대·흙침대의 온열 기능을 핵심 사용 이유로 꼽는 점에서 착안. 현재 등받이쿠션은 온열 기능 없이 물리적 지지만 제공하는데, 온열 기능을 결합하면 "돌침대 온열 + 등받이 지지 + 허리 온열" 3중 케어 구조로 프리미엄 포지셔닝 가능
- **형태:** 등받이쿠션 전면 패널에 장착 가능한 USB 저전력 발열 커버(40~45°C 저온 유지, 과열 방지 자동 차단) — 독립 판매 후 기존 제품에 덧씌우는 방식
- **기술 검토 포인트:** 발열 소재 안전인증(KC 인증) 필수 / USB 5V 저전력 설계로 안전성 확보
- **예상 가격:** 29,000~39,000원 단품 / 등받이쿠션 + 온열커버 패키지 70,000~80,000원
- **주의:** 전기용품 인증 절차 필요 → 개발 리드타임 3~6개월 예상, 내년 겨울 시즌(2026년 11월) 출시 타겟

### 아이디어 2: "따숨마루 미니 베개 (경추 지지형)" — 돌침대 취침용

- **근거:** 돌침대·흙침대 사용자는 취침 시 높이가 낮고 단단한 면 특성상 기존 일반 베개와 궁합이 맞지 않는다는 니즈가 존재. 등받이쿠션과 동일 소재·동일 브랜드로 경추 지지형 미니 베개를 출시하면 브랜드 일관성 강화 + 세트 구성 가능
- **형태:** 저반발 메모리폼 내장, 고·중·저 세 가지 높이 선택, 극세사 스웨이드 커버 동일 적용, 40cm×25cm 미니 사이즈
- **판매 방식:** 단품 + "돌침대 완전 셋업 패키지" (등받이쿠션 + 미니 베개 + 방석) 3종 세트
- **예상 가격:** 단품 35,000~45,000원 / 3종 세트 99,000~110,000원
