## 2. 기존 제품 개선 아이디어

### A-라인 개선

**라지(L) 사이즈 확장 버전 — "A-라인 와이드" 출시**

- **현황:** 현행 50cm×45cm 스펙이 평균 키 이하(155cm 내외) 고객에게는 최적이지만, 키 165cm 이상이거나 넓은 등·어깨 체형 고객에게는 지지 면적이 부족하다는 피드백이 누적됨
- **개선안:** 가로 60cm × 높이 50cm로 10cm씩 확장한 A-라인 와이드 SKU 신설 — 내부 폼 스펙(경도 유지)은 동일하게 가져가되 원단 재단·봉제 면적 확대
- **판매 전략:** 기존 A-라인을 "스탠다드" 표기로 유지, 와이드를 "L" 배지로 구분 → 상세페이지 내 사이즈 비교 표 제공으로 구매 결정 지원
- **원가 영향:** 원단 소요량 약 30% 증가 → 소재 원가 약 2,000~3,000원 증가 / 판매가 4,000~5,000원 인상 여력 확보
- **구현 난이도:** 낮음 (패턴·폼 사이즈만 변경, 기존 공장 라인 동일 적용)
- **우선 액션:** 생산처에 와이드 시제품 1개 의뢰, 165~170cm 신체 모델 착용 비교 촬영 후 내부 검토

### Cozy-라인 개선

**3단계 충전재 선택제 — "Cozy 소프트 / 미디엄 / 펌" 구분 출시**

- **현황:** 현재 Cozy-라인의 쿠션감은 단일 스펙으로 제공 중. 고객마다 선호 경도가 달라 "너무 푹신하다" vs. "딱 좋다"가 혼재 → 일부 고객은 구매 후 충전재 직접 조절 시도 사례 확인
- **개선안:** 동일 외형의 Cozy 커버에 충전재(폼 밀도 또는 충전 솜 양)를 3단계로 차등 적용 — ① Soft(현행 수준) ② Medium(폼 밀도 20% 증가) ③ Firm(폼 밀도 40% 증가, 허리 통증 집중 케어 타겟)
- **판매 방식:** 구매 시 경도 선택 옵션 추가 (옵션 항목 추가, 가격 동일 또는 Firm은 1,000원 추가)
- **상세페이지 표현:** "나의 등 타입에 맞게 고르세요 — 체형·통증 강도별 경도 가이드" 인포그래픽 제작
- **기대 효과:** 선택지 제공으로 구매 전환율 향상 + 반품·교환률 감소 (기대치 미스매치 최소화)
- **우선 액션:** 현 공장에서 Medium / Firm 시제품 각 1개 제작 의뢰 → 내부 착용 비교 테스트

## 3. 파생·변형 제품 아이디어

### "따숨마루 넥 필로우 (이동형 경추 쿠션)" — 외출·이동 중 허리·목 케어 라인

- **개요:** 기존 등받이쿠션이 돌침대·흙침대 위 '고정 사용' 형태인 반면, 타겟 고객(50대)이 자동차 이동 시간·병원 대기·공항 라운지 등 외부 환경에서도 동일한 척추 케어 욕구를 갖는다는 점에서 착안
- **기획 근거:**
  - 50대 타겟 고객은 정기적 병원 방문(한의원·척추 전문 병원) 동선이 잦음 → 이동 중 목·허리 불편 경험 빈번
  - 기존 등받이쿠션 리뷰에서 "차 안에서도 쓸 수 있나요?"라는 문의 패턴 반복 확인 (CS 피드백 참조)
  - 이동용 경추 쿠션 시장은 성장 중이나 "따숨마루 감성 + 소재"로 접근 시 차별화 충분
- **제품 스펙 (안):**
  - 말굽(U자) 형태, 30cm×28cm, 저반발 메모리폼 내장
  - 소재: 극세사 스웨이드 동일 (브랜드 통일성), 탈착 가능 커버
  - 컬러: 차콜 그레이 / 웜 베이지 (기존 라인업과 동일)
  - 수납 파우치 동봉 (이동 편의)
- **출시 전략:**
  - "돌침대 위에서 → 차 안에서도 따숨마루" 스토리라인으로 브랜드 사용 영역 확장
  - 스마트스토어 크로스셀링: 등받이쿠션 구매 고객에게 넥 필로우 할인 쿠폰 발행
  - 가격: 단품 22,000~28,000원 / 등받이쿠션 + 넥 필로우 세트 65,000~72,000원
- **리스크:** 넥 필로우 시장 경쟁이 치열 → 소재(극세사 스웨이드)와 브랜드 일관성만이 차별점. 인체공학 설계 데이터 기반 스펙 수립 필수

## 6. 시너지 제품 아이디어

### 아이디어 1: "따숨마루 돌침대 사이드 포켓 — 침대 옆 수납 솔루션"

- **근거:** 돌침대·흙침대 사용자는 침대 옆에 리모컨·안경·핸드폰·약 등 개인 소지품을 둘 안전한 공간이 부족하다는 공통 불편이 존재. 특히 50대 타겟은 취침 전 복용하는 건강기능식품·약이 많아 침대 옆 수납 니즈가 높음
- **형태:** 돌침대 매트리스·프레임에 걸치는 사이드 포켓 오거나이저 (패브릭 소재, 3~4개 포켓 구성), 극세사 스웨이드 동일 소재로 따숨마루 브랜드 톤 유지
- **판매 방식:** 단품 12,000~16,000원 / 등받이쿠션 구매 시 추가 구매 유도 쿠폰 제공
- **기대 효과:** 기존 고객 업셀링 채널 신설 + 돌침대 생활 전반을 커버하는 "라이프스타일 브랜드"로 인식 전환 기여

### 아이디어 2: "따숨마루 케어 파우치 — 세탁·보관 키트"

- **근거:** 리뷰에서 세탁 방법 문의·커버 관리법 불만이 꾸준히 접수됨. 세탁망 + 케어 가이드 카드 + 탈취 스프레이 미니어처를 세트로 구성한 "쿠션 케어 키트"를 동봉하거나 별도 판매하면 고객 불만을 선제 해소하면서 부가 수익 창출 가능
- **구성물:** 메쉬 세탁망(40cm×50cm) 1개 + 케어 가이드 카드(올바른 세탁·보관법) + 라벤더 패브릭 탈취 미니 스프레이(50mL)
- **판매 방식:** 등받이쿠션 구매 시 "케어 키트 추가 구성" 번들 옵션 (추가 5,000원) / 별도 단품 8,000원
- **기대 효과:** 세탁 관련 클레임·반품 감소 + 브랜드 애프터케어 인식 강화 → 재구매율 제고
