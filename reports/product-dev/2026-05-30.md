## 2. 기존 제품 개선 아이디어

### 🔷 A-라인 개선

**아이디어: 여름 전용 쿨링 커버 (시즌 에디션)**

- **문제:** 극세사 스웨이드 소재는 보온성은 우수하나 하절기 통기성 부족으로 땀 축적 → 사용 이탈 발생
- **개선안:** 기존 커버 위에 덧씌우거나 교체 가능한 '쿨링 메시 커버' 별도 출시. 소재: 에어쿨 메시(폴리에스터 고기능 편직) + 항균 처리. 색상은 기존 차콜·베이지 팔레트 유지
- **포지셔닝:** "A-라인 여름 에디션 커버" — 본품 소지자 대상 크로스셀 + 여름 신규 고객 진입 장벽 완화
- **기대 효과:** 계절성 판매 약세 구간(6~8월) 매출 방어, 여름 리뷰 품질 개선
- **개발 난이도:** ★★☆☆☆ (봉제 구조 동일, 소재만 교체)
- **예상 원가:** 커버 단품 원가 3,500~5,000원 / 판매가 15,000~18,000원 목표

---

### 🔶 Cozy-라인 개선

**아이디어: 측면 롤 보조쿠션 일체형 구조 (Side Bolster 통합형)**

- **문제:** 라운드 형태의 Cozy-라인은 풍성한 쿠션감은 호평이나, 좌우 지지 없이 몸이 기울면 쿠션 전체가 이동하는 구조적 약점 존재
- **개선안:** 본체 양 옆에 소형 롤형 볼스터를 일체화한 'Wing-Cozy' 구조. 볼스터는 탈착 가능(벨크로)하여 필요 시 제거, 팔받침 역할도 겸함
- **차별점:** 단순 쿠션 → '좌식 힐링 체어' 개념으로 업그레이드, 사진 비주얼 임팩트 강화
- **기대 효과:** 불만 2위(측면 지지 부재) 직접 해소, SKU 프리미엄화로 가격 저항 완화
- **개발 난이도:** ★★★★☆ (금형·봉제 구조 변경, 샘플 제작 2~3회)
- **예상 판매가 상승:** 기존 대비 +12,000~18,000원 프리미엄 포지셔닝 가능

## 3. 파생·변형 제품 아이디어

### 🆕 따숨마루 키즈·주니어 라인 — 청소년 자세 교정 등받이 쿠션

**컨셉:** 40~60대 메인 고객의 자녀·손자녀를 위한 '성장기 자세 교정' 등받이 쿠션

- **배경:** 기존 구매자 리뷰에서 "자녀도 쓴다", "학생 자녀 공부방에 뒀더니 잘 쓴다"는 언급이 소수지만 반복적으로 등장. 가족 내 추천 패턴과 결합하면 패밀리 라인 확장 명분 충분
- **차별점:**
  - 크기: A-라인 대비 20% 소형(청소년 체형 맞춤)
  - 기능: 척추 측만 예방용 경사 가이드라인(내부 구조) + 앉는 자세 안내 인쇄물 동봉
  - 소재: 기존 극세사 스웨이드 + 항균·진드기 차단 내피 (학습 공간 위생 강조)
  - 색상: 코랄 핑크 / 민트 그레이 (청소년 감성 확장)
- **타겟:** 따숨마루 기존 구매자의 자녀(중·고등학생) + 자녀 교육 관심 40~50대 학부모
- **크로스셀 포인트:** 부모·자녀 세트 구성 → "온 가족 허리 케어" 메시지
- **예상 판매가:** 38,000~48,000원 (A-라인 대비 약간 낮게 포지셔닝)
- **마케팅 포인트:** "돌침대에서 공부방까지, 따숨마루가 지키는 가족 척추 건강"

  - 등받이 쿠션 사용 후 온열 팩으로 마무리하는 허리 케어 루틴 자연스럽게 연결
  - OEM 원가 낮고 브랜드 로고 인쇄 용이 (최소 발주량 200개)
- **사용 시나리오:** "따숨마루 쿠션으로 바르게 앉고 → 스트레칭 후 온열 팩으로 마무리" 루틴 콘텐츠화
- **예상 매입가:** 3,500~5,000원 / 판매가: 15,000~22,000원
- **스마트스토어 번들 전략:** 등받이 쿠션 + 냉온열 팩 세트 "따숨마루 허리 케어 세트" → 기프트 포지셔닝으로 객단가 극대화

## 6. 시너지 제품 아이디어

### 시너지 제품 1: 따숨마루 침대 트레이 테이블

- **컨셉:** 돌침대 위에서 책·태블릿·스마트폰을 올려두고 사용하는 이동형 bamboo 트레이 테이블
- **소재:** 천연 대나무(항균·내습) + 논슬립 실리콘 패드 하단
- **사용 시나리오:** 등받이 쿠션에 기댄 채 트레이 위에서 독서·태블릿 시청·음식 섭취 → "돌침대 위 나만의 힐링 공간" 구현
- **시너지 이유:** 따숨마루 구매자가 쿠션에 기댄 채 가장 많이 하는 행동(독서·핸드폰)을 직접 보조, 자연스러운 업셀
- **컬러:** 내추럴 우드 + 블랙 스틸 다리 (미니멀 감성)
- **예상 판매가:** 35,000~55,000원

### 시너지 제품 2: 따숨마루 취침 전 아로마 디퓨저 세트

- **컨셉:** 돌침대 침실 분위기 연출을 위한 초소형 우드 아로마 디퓨저 + 라벤더·유칼립투스 블렌드 오일 세트
- **소재:** 천연 우드 바디 + USB 초음파 진동식
- **시너지 이유:** 돌침대·흙침대 사용 = 건강 중시 라이프스타일 → 수면 질 개선·이완 용품으로 니즈 자연 연결
- **기프트 패키지:** 등받이 쿠션 + 아로마 세트 "따숨마루 수면 웰니스 선물 세트" → 명절·어버이날 프리미엄 선물 포지셔닝
