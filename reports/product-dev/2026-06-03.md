## 2. 기존 제품 개선 아이디어

### A-라인 개선안 — 미끄럼 방지 + 허리 서포트 업그레이드

**현재 문제:** 돌침대·흙침대는 표면이 매끄러운 원적외선 석재 → 쿠션이 미끄러져 사용 중 위치 이탈 빈번

**개선 방향:**
- **후면 전체 실리콘 도트 논슬립 패드** 적용 (기존 패브릭 후면 → 논슬립 실리콘 격자 패턴으로 교체)
- 전면 패널 수평 봉제선을 1줄 → 2줄로 변경하여 **요추 지지 존(Zone) 시각적 구분** 및 내부 심재(폼) 이중 밀도 구성 (상단 소프트 / 하단 서포트)
- 예상 효과: 위치 이탈 방지 + 장시간 착석 시 허리 피로감 감소
- 추가 원가 예상: 소재 비용 +8~12% (실리콘 도트 소재), 심재 이중 구성 +5~8%

### Cozy-라인 개선안 — 분리 세탁 커버 + 목 지지 기능 추가

**현재 문제:** 풍성한 쿠션감이 강점이나 세탁 용이성 낮음 → 위생 우려로 장기 사용 기피

**개선 방향:**
- **YKK 지퍼 탈착식 외피 커버** 도입 (극세사 스웨이드 커버 → 분리 가능한 이중 커버 구조)
  - 내피: 기존 스웨이드 고정 / 외피: 30°C 세탁 가능 면혼방 커버 (차콜·베이지 각 2종)
- **상단 목베개 일체형 볼스터** 소형 추가 (쿠션 상단에 U자형 소형 넥롤 탈부착 벨크로 연결)
  - 돌침대에 앉아 TV 시청 시 목·머리까지 지지되는 올인원 경험 제공
- 예상 효과: 재구매율↑, 가족 선물 세트 구성 가능
- 추가 원가 예상: 지퍼 + 이중 커버 +15%, 볼스터 추가 +20% → 세트 프리미엄 가격 책정으로 마진 유지 가능

## 3. 파생·변형 제품 아이디어

### 따숨마루 돌침대 전용 다리 받침 쿠션 (레그 리프터)

**배경 분석:**
- 돌침대 사용자의 핵심 불편 중 하나가 **누웠을 때 다리·무릎 밑 공간 지지 부족**
- 허리에 가해지는 압력을 분산하려면 무릎 밑 쿠션이 필수 (요추 곡선 유지)
- 현재 시장에 돌침대 전용 레그 쿠션 거의 전무 → 블루오션

**제품 컨셉:**
- 형태: 완만한 아치형 원통 레그 볼스터 (지름 18cm × 길이 50cm)
- 소재: 기존 A·Cozy-라인과 동일한 **극세사 스웨이드 (차콜 / 베이지)** — 브랜드 통일성
- 내부: 고밀도 메모리폼 (온열 사용 인증 규격)
- 기능: 무릎 밑 받침 → 요추-골반 압박 감소, 하지정맥류 예방 효과
- 포지셔닝: "등받이 쿠션과 함께 쓰는 따숨마루 세트" → **세트 판매 패키지** 구성으로 객단가 상승
- 예상 판매가: 29,000~35,000원 / 세트 구성 시 쿠션+레그볼스터 = 10~15% 할인 번들

## 5. 소싱 추천 제품

### 따숨마루 공식 인증 돌침대 전용 토퍼 패드 (PB 소싱 or 협업)

**소싱 배경:**
- 돌침대 사용자의 1위 니즈: 온열은 살리면서 딱딱함 완화
- 현재 시장의 일반 토퍼는 고온 변형·라돈 미검증 제품 다수 → 불신 팽배
- 따숨마루가 **"돌침대 라이프 스타일 브랜드"**로 확장 시 자연스러운 카테고리

**소싱 스펙 제안:**
- 두께: 5cm (온열 효과 유지 + 쿠션감 균형 최적 두께)
- 소재: 온열기구 사용 인증 메모리폼 + 극세사 스웨이드 커버 (기존 쿠션 소재 통일)
- 인증: 온열기구 사용 가능, 라돈 불검출 시험성적서 필수
- 색상: 차콜 그레이 / 웜 베이지 (기존 라인 통일)
- 예상 소싱 단가: 25,000~40,000원 / 판매가 75,000~99,000원
- **브랜드 가치:** 단순 쿠션 판매자 → 돌침대 라이프 전문 브랜드로 포지셔닝 격상

## 6. 시너지 제품 아이디어

### 아이디어 A — 따숨마루 돌침대 전용 목베개 (넥 롤 필로우)

- 돌침대에서 TV 시청·독서 시 목이 꺾이는 문제 해결
- 형태: U자형 반월 넥롤 (지름 12cm × 길이 40cm)
- 소재: 기존 극세사 스웨이드 통일 → 브랜드 패밀리 룩 형성
- 판매 전략: 등받이 쿠션 구매자 전용 추가 구매 할인 (10~15%) → 재구매·업셀링
- 예상 판매가: 19,000~25,000원

### 아이디어 B — 따숨마루 침대 옆 독서대·태블릿 거치대

- 돌침대 사용자가 등받이 쿠션 기댄 채로 스마트폰·태블릿·책 보는 수요 확인
- 형태: 대나무 접이식 침대 트레이 (무릎 위 거치)
- 브랜드 연계: 따숨마루 차콜·베이지 패브릭 파우치 증정 세트 구성
- 포지셔닝: "돌침대 위의 나만의 쉼터 세트" — 등받이쿠션 + 레그볼스터 + 독서대 3종 풀세트
- 예상 판매가: 35,000~45,000원 (세트 시 패키지 가격 책정)

## 7. 이번 주 기획·개발 우선순위

### Priority 1 — A-라인 논슬립 패드 샘플 발주 및 테스트 (완료 목표: 이번 주 금요일)

- 실리콘 도트 논슬립 소재 국내 소싱처 3곳 견적 요청
- 실제 돌침대 표면(석재·세라믹) 대상 미끄럼 방지 성능 비교 테스트
- 기존 A-라인 후면 교체 시 원가 변동 시뮬레이션 완료

### Priority 2 — Cozy-라인 지퍼 탈착식 커버 구조 설계

- 지퍼 위치 및 탈착 편의성 목업 제작 (패턴 수정)
- 외피 소재 선정: 30°C 세탁 가능 면혼방 스웨이드 후보군 3종 테스트
