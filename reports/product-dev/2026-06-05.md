## 2. 기존 제품 개선 아이디어

### A-라인 (삼각 웨지, 심플 고급 디자인)

**아이디어 A-1: 각도 조절 인서트 시스템**
- 내용: 쿠션 내부에 두께 다른 웨지 인서트 2종(표준 15° / 높음 20°)을 교체 삽입 가능한 포켓 구조 도입
- 효과: 체형·허리 상태에 따른 맞춤 각도 제공 → 불만 TOP2 직접 해소
- 구현 방식: 후면 지퍼 개구부 + 인서트 2종 동봉, 추가 인서트 별도 판매 가능
- 예상 비용 증가: 약 8~12%

**아이디어 A-2: 항균 이지케어 이중 커버**
- 내용: 겉 커버(기존 극세사 스웨이드 유지) 안쪽에 항균 방오 코팅 속커버를 별도 레이어로 구성
- 효과: 속커버만 분리해 가정 세탁기 세탁 가능, 겉 커버는 드라이클리닝 주기 연장
- 소재 제안: 속커버 — 항균 처리 폴리에스터 니트 (세탁 200회 이상 내구성)
- 마케팅 포인트: "돌침대 위생을 가장 쉽게"

### Cozy-라인 (라운드, 포근한 쿠션감)

**아이디어 C-1: 허리 집중 지지 내부 코어 추가**
- 내용: 풍성한 외형은 유지하되 내부 허리 접촉 구간(하단 20cm)에 고탄성 메모리폼 코어 삽입
- 효과: 포근한 느낌 + 요추 지지 강화 → "쿠션감은 있는데 허리가 오히려 빠진다"는 일부 지적 해소
- 소재: 고밀도 메모리폼(밀도 50D 이상) + 기존 폴리에스터 충전재 혼합 구조
- 마케팅 포인트: "편안함과 지지력, 두 마리 토끼"

**아이디어 C-2: 사이드 핸들 + 벽걸이 고리 추가**
- 내용: 쿠션 상단에 가죽 루프 핸들 1개, 후면에 벽걸이 후크홀 추가
- 효과: 돌침대 사용 특성상 침대 옆 세워두거나 이동이 잦음 → 수납·이동 편의성 개선
- 부가 효과: 핸들이 포토제닉 요소로 작용 → 인스타그램 노출 자연 증가
- 비용 영향: 미미 (가죽 루프 원가 500~800원 수준)

## 3. 파생·변형 제품 아이디어

### 「따숨마루 슬림 요추쿠션」— 돌침대 전용 얇은 허리 서포터

- **콘셉트:** 등받이 쿠션과 함께 사용하는 별도의 **슬림형 요추 받침 쿠션** (두께 6~8cm, 폭 40cm)
- **배경:** 등받이 쿠션 사용자 중 "등은 편한데 허리 밑 공간이 뜬다"는 불만 다수 → 기존 제품의 상위 기능만으로 커버되지 않는 허리-침대 사이 빈 공간 해소
- **디자인:** 기존 A-라인과 동일한 극세사 스웨이드 소재, 차콜/베이지 색상 통일 → 세트 구성으로 번들 판매 가능
- **가격 전략:** 단품 ₩29,000 / A-라인 + 슬림요추쿠션 세트 ₩89,000 (15% 번들 할인)
- **핵심 차별점:** "등받이 + 요추" 2단 지지 시스템 — 돌침대 특화 첫 조합 상품

## 5. 소싱 추천 제품

### 「고반발(고탄성) 하이브리드 폼 — 돌침대 전용 쿠션 충전재 업그레이드」

- **제품 사양:** 고반발 폼(탄성 60% 이상) + 메모리폼 혼합 레이어드 시트 (두께 5cm 기준)
- **소싱 이유:**
  - 현 극세사 스웨이드 외피는 유지하되 내충전재를 업그레이드 → 원가 대비 체감 품질 대폭 향상
  - 고반발폼은 돌침대의 단단한 면에 올려도 눌림 없이 체중 분산 → 돌침대 전용 특성과 정합
  - 메모리폼 대비 복원력 빠름 → 오래 써도 쿠션 형태 유지 (내구성 불만 사전 예방)
- **예상 소싱처:** 국내 폼 전문 제조사 (경기 포천·화성 산단 다수), MOQ 500m² 수준
- **원가 예상 증가:** 단품 기준 ₩2,500~₩4,000 내외
- **권장 액션:** 샘플 3종(고반발 단독 / 메모리폼 단독 / 하이브리드) 발주 후 내부 블라인드 테스트

## 6. 시너지 제품 아이디어

### 시너지 ①: 「따숨마루 돌침대 전용 무릎 쿠션」

- **콘셉트:** 돌침대에 등받이 쿠션 기대어 앉을 때 무릎 아래 공간을 받쳐주는 원형/반원형 소형 쿠션
- **필요성:** 딱딱한 돌침대에서 다리를 뻗거나 무릎 굽혀 앉을 때 무릎·발목 압박 발생 → 등받이 쿠션 구매 고객의 자연 연속 구매 유도
- **가격 포인트:** ₩19,000~₩24,000 / 등받이 쿠션과 세트 구성 시 ₩10,000 할인
- **색상:** 기존 차콜 그레이 / 웜 베이지 동일 적용 → 세트 비주얼 완성도 극대화

### 시너지 ②: 「따숨마루 온열 패드 (돌침대 위 등 찜질용)」

- **콘셉트:** 등받이 쿠션 전면 패널에 부착 또는 별도 사용 가능한 USB 온열 패드 (40°C 저온 온열)
- **필요성:** 돌침대 사용자 = 온열 건강 관심 높음 → 돌침대의 온열 기능을 등 부위까지 확장
- **타겟 연계:** 등 통증 + 냉증 있는 50대 여성 — 겨울·환절기 수요 매우 높을 것으로 예상
- **기술 수준:** USB 타입 온열 패드 OEM 다수 존재 → 소싱 난이도 낮음
- **주의사항:** 전자파·안전인증(KC) 필수 확인

## 7. 이번 주 기획·개발 우선순위

### Priority 1 — A-라인 각도 조절 인서트 시스템 프로토타입 제작 착수
- **이유:** 고객 불만 TOP2(높이 조절 불가)를 직접 해결, 경쟁사 미보유 기능 → 특허·디자인 등록 확장 가능성
- **이번 주 목표:** 인서트 두께 2종(3cm / 5cm) 스펙 확정 + 내부 포켓 구조 패턴 도면 작성
- **담당 액션:** 소싱팀에 고반발폼 인서트 샘플 3종 발주 요청

### Priority 2 — 「슬림 요추쿠션」신제품 기획서 초안 작성
- **이유:** 번들 판매를 통한 객단가 향상(+₩29,000) + 기존 A-라인 고객 재구매 유도
- **이번 주 목표:** 제품 스펙 확정(사이즈·소재·충전재) + 예상 원가 산출 + 스마트스토어 등록 타임라인 설정
- **담당 액션:** 기존 극세사 스웨이드 원단 잔량 확인, 슬림 패턴 제작 의뢰

### Priority 3 — 이지케어 커버 소재 테스트 및 공급처 조사
- **이유:** 고객 불만 TOP1(커버 탈부착 불편), TOP3(세탁 어려움) 동시 해소 → CS 비용 절감 효과
- **이번 주 목표:** 항균 코팅 니트 원단 공급사 2~3곳 견적 취합 + 기존 커버 패턴에 이중 레이어 적용 가능성 검토
- **담당 액션:** 동대문 원단 시장 또는 온라인 B2B 플랫폼(패브릭타임·원단나라) 통해 샘플 요청
