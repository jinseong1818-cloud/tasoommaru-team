## 2. 기존 제품 개선 아이디어

### A-라인 개선: "탈착식 항습 거치 받침대 동봉"

- **문제:** 장마철 습기 문의가 화요일에도 지속 — 제품 자체 결함이 아닌 "보관 방법 부재"가 근본 원인
- **제안:** 접이식 메시 거치대(스탠드)를 옵션 또는 기본 동봉품으로 제공 — 평소엔 거치대에 세워 보관, 사용 시 펼쳐서 사용. 통기 구조로 바닥 접촉면 습기 축적 차단
- **기대 효과:** 장마철 CS 문의량 감소 + "보관까지 설계된 제품"이라는 신뢰 강화 메시지로 전환
- **구현 난이도:** 하~중 (액세서리 소싱, 본품 구조 변경 없음 — 5번 소싱 추천과 연계)

### Cozy-라인 개선: "여름 전용 교체형 메시 커버" 별도 판매

- **문제:** 풍성한 라운드 형태 특성상 본품 구조를 바꾸지 않고도 여름철 통기성을 높이고 싶은 니즈
- **제안:** 기존 극세사 스웨이드 커버 대신 끼워 신는 에어메시 커버를 별도 옵션으로 판매 (본품 인서트는 그대로 유지, 커버만 교체)
- **기대 효과:** 본품 금형·내충재 변경 없이 여름 매출 방어 — 원가 부담 최소화하며 "4계절 커버 교체형"으로 신규 셀링 포인트 확보
- **구현 난이도:** 중 (커버 패턴 별도 제작 필요하나 내부 코어는 공용)

- **컨셉:** 기존 A-라인 디자인을 유지한 75% 축소 사이즈, 사무실 의자·차량 시트에 고정 가능한 벨크로/스트랩 부착

- **예상 판매가:** ₩42,000~52,000
- **우선순위:** ★★★★★ — 영업/CS 데이터로 이중 검증된 아이디어이므로 이번 주 샘플 기획 착수 권장 (7번 참조)

- **예상 소싱가:** 개당 ₩3,500~5,500 (MOQ 200개 기준, 국내 소싱)

## 7. 이번 주 기획·개발 우선순위

### Priority 1 ★★★★★ — 장마철 통풍 거치대 샘플 소싱 (시급)

- **배경:** 장마가 진행 중이며 CS팀 보관 문의가 매일 발생 — 시즌 내 출시해야 효과가 있는 시급성 높은 아이템
- **액션:**
  1. 국내 생활용품 부자재 제조사 2~3곳에 절첩식 메시 거치대 샘플 요청 (이번 주 수요일까지)
  2. CS팀과 협업해 "세워서 보관" 안내 콘텐츠를 제품 동봉 카드로 전환
  3. 장마철 한정 "보관 세트" 번들가 시뮬레이션
- **목표:** 6월 말 샘플 확정 → 7월 초 옵션 상품 등록

### Priority 2 ★★★★★ — 오피스·카시트 미니 라인 개발 착수

- **배경:** 제품기획팀(Travel-라인)과 CS팀(캠페인) 양쪽에서 독립적으로 동일 수요 신호 포착 — 교차 검증된 최우선 아이템
- **액션:**
  1. 기존 제조 파트너사에 75% 축소 샘플 + 벨크로/스트랩 고정 구조 제작 요청
  2. CS팀 캠페인 반응(문의·전환) 데이터를 1주일 단위로 공유받아 수요 강도 재확인
  3. 예상 판매가·번들 구성안 작성
- **목표:** 7월 중 샘플 수령, CS팀 캠페인과 동시 출시 목표

### Priority 3 ★★★★☆ — 상세페이지 미해결 이슈 에스컬레이션

- **배경:** 스마트스토어팀이 여러 보고서에 걸쳐 "특허·디자인 등록 배지 노출 여부", "A-라인/Cozy-라인 비교 Q&A" 미확인 상태를 반복 지적 중 — 정품 신뢰 관련 CS 문의와도 직결된 사안
- **액션:**
  1. 담당자(스마트스토어 운영) 직접 컨택해 실제 페이지 반영 여부 확인 요청 — AI 보고서로는 더 이상 확인 불가하므로 사람이 직접 점검 필요
  2. 색상 옵션 확대(시즌 한정 컬러) 타당성 검토 — 디자인팀과 소량 샘플 논의 시작
- **목표:** 이번 주 내 사람 담당자의 실제 확인 완료, 미해결 시 명확한 원인 파악
