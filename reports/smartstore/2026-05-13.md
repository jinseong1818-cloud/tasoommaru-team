## 라인별 타겟 전략

### A-라인 강조점 (웨지 구조·간결한 디자인)
- **핵심 소구**: "삼각형 웨지 구조로 등·허리를 자연스러운 각도로 지지"
- **강조 비주얼**: 측면 단면 구조 컷 → 경사각 시각화
- **타겟 메시지**: "오래 앉아도 허리가 편한 이유가 있습니다"
- **추천 키워드**: `돌침대등받이`, `허리각도쿠션`, `웨지쿠션`, `등받이쿠션딱딱한바닥용`
- **색상 강조**: 차콜 그레이(남성·모던 감성) / 웜 베이지(여성·따뜻한 감성) 옵션 분리 노출 권장

### Cozy-라인 강조점 (풍성한 쿠션감·포근함)
- **핵심 소구**: "폭신하게 감싸는 라운드형, 장시간 사용에도 꺼지지 않는 쿠션감"
- **강조 비주얼**: 쿠션에 기대는 모델 사이드샷 → 포근함 연출
- **타겟 메시지**: "딱딱한 돌침대 위에서도 소파처럼 편안하게"
- **추천 키워드**: `포근한등받이쿠션`, `시니어쿠션`, `흙침대쿠션`, `돌침대소파쿠션`
- **공통 강조**: 특허·디자인 등록 배지 → 상품 이미지 우측 상단에 뱃지 삽입 제안

> 페이지 직접 확인 불가로 일반적 구성 기준 개선안 제시 (다음 수집 시 실제 반영 여부 검토 필요)

2. **USP 배너 (상단 3개 이내)**:
   - "돌침대·흙침대 전용 설계" → 일반 쿠션과 차별화 즉시 인지
   - "특허 및 디자인 등록" → 신뢰성 배지
   - "리뷰 150개↑ 실사용 후기" → 사회적 증거

3. **구조 설명 섹션**: A-라인 단면 도식 + 각도 수치 시각화
   - "몇 도 경사" 수치화 → 구체성이 신뢰감 형성

4. **소재 클로즈업**: 극세사 스웨이드 질감 매크로 컷 → 프리미엄 감 강조

5. **리뷰 섹션**: 허리 통증 언급 리뷰 상단 고정 (타겟 공감 극대화)
