## 1. 유튜브 쇼츠 아이디어 (60초 이내)

### 제목: "돌침대에서 허리 펴는 법 — 대부분이 모르는 1가지"

**훅 (0~3초)**  
> "돌침대 쓰는데 아침마다 허리가 아프세요? 이유가 있습니다."

**핵심 장면 구성 (3~50초)**
| 구간 | 장면 |
|------|------|
| 3~10초 | 돌침대 위 일반 자세 vs 등받이 쿠션 사용 자세 비교 컷 |
| 10~25초 | 삼각형 인체공학 쿠션이 허리 곡선에 밀착되는 클로즈업 |
| 25~40초 | "앉아서 TV 볼 때", "스마트폰 볼 때" 실제 사용 상황 2컷 |
| 40~50초 | 사용 전·후 표정 비교 (과장 없이, 편안함 강조) |

**CTA (50~60초)**  
> "링크에서 특허 등록 제품 확인해 보세요. 돌침대·흙침대 전용으로 설계됐어요."  
> 화면 하단: 스마트스토어 링크 + "저장해 두세요 👆" 텍스트 오버레이

**제작 팁**
- BGM: 잔잔한 자연음 (새소리, 바람소리) — 시니어 친화적
- 자막: 크고 굵은 폰트, 노란/흰 배경 자막으로 가독성 확보
- 화면 전환: 빠른 컷 편집 지양, 2~3초 호흡 유지

## 2. 일반 영상 아이디어 (5~10분)

### 제목: "돌침대·흙침대 사용자가 허리 편하게 앉는 5가지 자세 — 전문가가 알려드려요"

**타겟 키워드:** 돌침대 허리 통증, 흙침대 바른 자세, 시니어 허리 건강, 등받이 쿠션 추천

**목차**
1. **인트로 (0:00~0:45)** — 돌침대·흙침대 위에서 오래 앉으면 왜 허리가 불편한지 설명
2. **챕터 1 (0:45~2:00)** — 딱딱한 침대면이 허리에 미치는 영향 (도해 또는 모형 활용)
3. **챕터 2 (2:00~4:00)** — 잘못된 자세 3가지 시연 (실제 영상, 짧게)
4. **챕터 3 (4:00~6:30)** — 올바른 5가지 자세 시연 (등받이 쿠션 활용 포함)
5. **챕터 4 (6:30~8:00)** — 따숨마루 쿠션 소개: 삼각형 설계·특허 포인트 자연스럽게 연결
6. **아웃트로 (8:00~9:00)** — 시청자 질문 유도 + 링크 안내

**연출 가이드**
- 출연자: 40~60대 실제 사용자 또는 전문 강사 섭외 권장
- 화면 구성: 자막 + 인포그래픽 병행, 자세 교정 장면은 측면 앵글 필수
- 중간 광고 삽입 지점: 챕터 3 시작 전 (4:00)

## 3. 썸네일 전략

### 핵심 원칙
- **시니어 가독성 우선:** 텍스트 최소화(3단어 이내), 폰트 크기 72pt 이상
- **감정 포착:** 찡그린 표정 → 편안한 표정 대비 구성 (Before/After 없이 After만 보여주는 것도 효과적)
- **색상 전략:** 따뜻한 베이지·갈색 계열(제품 연상) + 강조색 노란/주황

### 쇼츠 썸네일 시안
```
[배경: 따뜻한 나무 느낌 베이지]
  중앙: 쿠션 위에 앉은 편안한 표정의 60대 모델
  상단 텍스트: "이거였어요" (굵은 흰 글씨)
  하단 텍스트: "돌침대 허리 통증" (노란 강조 배경)
```

### 일반 영상 썸네일 시안
```
[좌측 50%]: 쿠션 클로즈업 (제품 실물)
[우측 50%]: 모델 옆모습 (올바른 자세)
[상단 바]: "돌침대 허리 5가지 자세" (흰 글씨 + 진한 배경 스트립)
[우하단 뱃지]: "특허 등록" 스탬프 아이콘
```

## 4. 영상 제목 SEO 최적화 제안 3가지

> **금칙어 제외 원칙 적용** (치료, 완치, 의학적 효과, 업계 1위, 최고, 무조건, 100%, 절대, 완벽)

### 안 ① — 문제 해결형 (검색 의도 직접 타겟)
```
돌침대 쓰면서 허리 아픈 분 보세요 — 자세 하나로 달라집니다 | 따숨마루
```
- 타겟 키워드: `돌침대 허리`, `돌침대 통증`
- 감성 훅: "아픈 분 보세요" → 클릭 유도

### 안 ② — 정보성 + 신뢰 강화형
```
[흙침대·돌침대 전용] 허리 편한 등받이 쿠션 — 특허 받은 삼각형 인체공학 설계
```
- 타겟 키워드: `흙침대 등받이 쿠션`, `돌침대 전용 쿠션`
- 신뢰 요소: "특허", "인체공학" 명시

### 안 ③ — 시니어 공감형 (감성 어프로치)
```
어머니 돌침대 위에서 TV 보시나요? 허리 걱정 줄여드리는 방법
```
- 타겟 키워드: `시니어 허리`, `부모님 건강 선물`
- 2차 타겟: 부모님 선물을 찾는 30~40대 자녀 세대
