## 일반 영상 아이디어

**주제:** 돌침대·흙침대 위에서 올바르게 앉는 법 — 전용 쿠션이 필요한 이유

**제목:**
"돌침대 위에서 이렇게 앉고 계셨나요? 50대 허리가 무너지는 자세 TOP 3 + 올바른 세팅법 [따숨마루 등받이쿠션]"

**목차:**
1. 인트로 — 돌침대·흙침대를 쓰면서 허리가 오히려 불편해진 분들께 (공감 확보)
2. 돌침대 위에서 흔히 하는 잘못된 자세 TOP 3 (벽에 등 바로 기대기 / 옆으로 비스듬히 앉기 / 베개 뒤로 받치기)
3. 왜 일반 쿠션이나 베개로는 해결이 안 되는가 — 돌침대 전용 설계의 필요성
4. 따숨마루 등받이쿠션 소개 — 삼각형 웨지 구조와 특허·디자인 등록의 의미
5. 올바른 세팅법 실연 — 벽 지지 방법과 앉는 각도 (A-라인 vs Cozy-라인 비교)
6. 마무리 — 고객 리뷰 소개 + 스마트스토어 링크 안내

**타겟 키워드:**
돌침대 등받이쿠션, 흙침대 허리쿠션, 돌침대 올바른 자세, 허리 등받이쿠션, 좌식 허리 자세, 50대 허리 건강, 돌침대 전용쿠션

## 썸네일 전략

**핵심 컨셉:** Before/After 대비형 — 허리 각도의 시각적 차이

**코어 구성 요소:**

- **이미지 구도**: 화면을 좌우로 분할 — 왼쪽(Before)은 구부정하게 기댄 자세, 오른쪽(After)은 쿠션 사용 후 반듯한 자세. 배경은 동일한 돌침대 환경으로 통일감 유지.
- **제품 클로즈업 인서트**: 하단 중앙에 차콜 그레이 따숨마루 쿠션 단독 컷을 작게 배치 — 제품 인지도 강화
- **텍스트 (2줄)**: 상단 소형 "돌침대 위에서 이렇게 앉고 계셨나요?" / 하단 대형 볼드 강조 "자세가 허리를 결정합니다" — 문제 제기형 문구로 40~60대 클릭 심리 자극
- **컬러 팔레트**: 크림·웜베이지 배경 + 차콜 그레이 쿠션 강조 + 소형 붉은 원형 배지에 "돌침대 전용" 표기
- **인물**: 50대 초반 여성 모델이 오른쪽(After)에 편안한 미소로 등장 — 타겟 고객 자기동일시 유도
- **썸네일 A/B 후보**: A안(Before/After 인물 대비) vs B안(제품 단독 + 강조 카피) — 유튜브 스튜디오 A/B 테스트 적용 권장

1. **커뮤니티 탭 투표 게시물 업로드**
   - 질문: "돌침대 위에서 TV 보실 때 어떻게 앉으세요?"
   - 선택지 A: "벽에 등 바로 기댄다" / B: "베개를 뒤에 받친다" / C: "옆으로 비스듬히" / D: "쿠션 따로 씀"
   - 목적: 시청자 자발적 참여 유도 → 댓글 알림으로 재방문 증가 → 오늘 일반 영상으로 자연스러운 유입 연결

2. **댓글 상호작용 시나리오**
   - 허리 불편 경험 댓글 → "정말 공감이 가요! 혹시 돌침대·흙침대 사용 중이신가요? 저희 영상에 딱 맞는 세팅법 있어요 😊" 로 부드럽게 영상 연결
   - 제품 문의 댓글 → 상세한 A-라인/Cozy-라인 차이 답변 후 스마트스토어 링크 안내

3. **쇼츠 → 일반 영상 퍼널 연결**
   - 오늘 쇼츠 영상 설명란에 일반 영상 링크 고정 핀 댓글로 달기
   - 쇼츠 마지막 화면에 일반 영상 엔드카드 연결 — 시청 시간 지표 개선 효과

4. **누적 리뷰 활용**
