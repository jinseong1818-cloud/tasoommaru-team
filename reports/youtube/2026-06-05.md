## 쇼츠 아이디어

**주제:** 소파에 파묻혀 TV 볼 때 허리가 무너지는 순간 — 따숨마루로 소파 자세 교정

**훅 (0~3초):**
"소파에서 TV 보다가 일어날 때 허리 잡아보신 적 있으세요?"

**핵심 장면:**
1. 소파에 등받이 없이 푹 파묻혀 앉아 등이 C자로 굽은 채 리모컨을 드는 모습 (Before — 공감 유발, 옆면 클로즈업)
2. 따숨마루 A-라인 쿠션을 소파 시트와 등판 사이에 안정적으로 끼워 넣는 세팅 장면 — 손이 쿠션을 밀어 넣는 순간 클로즈업
3. 쿠션이 소파 등판에 단단히 고정된 채 등을 자연스럽게 받쳐주는 모습 — 여성의 허리와 어깨가 펴지고 표정이 부드럽게 풀림 (After)

**CTA:**
"소파도, 돌침대도 따숨마루 하나로 해결! 프로필 링크에서 확인하세요 👇"

## 일반 영상 아이디어

**주제:** A-라인 vs Cozy-라인 완벽 비교 — 내 생활에 맞는 따숨마루 선택법

**제목:**
"A-라인이냐 Cozy-라인이냐? 따숨마루 등받이쿠션 두 가지를 직접 비교해드립니다 [돌침대·소파 전용]"

**목차:**
1. 인트로 — "어떤 걸 골라야 하나요?" 구매 전 가장 많이 오는 질문에 답하다
2. A-라인 구조 상세 소개 — 삼각형 웨지 완만한 경사, 차콜 그레이·웜 베이지, 전면 수평 봉제선, 심플하고 고급스러운 디자인
3. Cozy-라인 구조 상세 소개 — 풍성한 쿠션감의 라운드 형태, 포근하고 감싸는 느낌, 동일 소재·색상
4. 장면별 비교 실연 ① 돌침대·흙침대 위에서 — 어떤 라인이 더 잘 맞나?
5. 장면별 비교 실연 ② 소파에서 — 각각의 밀착감 차이 직접 확인
6. 어떤 사람에게 A-라인이 맞고 어떤 사람에게 Cozy-라인이 맞는가 — 체형·선호도·생활환경 기준 정리
7. 마무리 — 특허·디자인 등록 공통 강점 정리 + 스마트스토어 링크 CTA

**타겟 키워드:**
따숨마루 A라인 Cozy라인 비교, 등받이쿠션 선택법, 돌침대 쿠션 추천, 허리쿠션 종류 차이, 소파 등받이쿠션, 흙침대 쿠션 추천, 50대 허리쿠션 고르는 법

## 썸네일 전략

**핵심 컨셉:** 나란히 비교형 — 두 라인의 실루엣 차이를 직관적으로 보여주는 "선택의 순간"

**코어 구성 요소:**

- **이미지 구도**: 화면 중앙에 A-라인(차콜 그레이)과 Cozy-라인(웜 베이지) 쿠션을 나란히 배치 — 두 쿠션의 옆면 실루엣이 명확히 대비되도록 측면 촬영. 배경은 크림 화이트 미니멀 인테리어.
- **좌우 라벨 텍스트**: 쿠션 각각 위에 "A-라인 (슬림·심플)" / "Cozy-라인 (포근·풍성)" 라벨을 볼드 흰 텍스트로 배치
- **상단 대형 질문형 문구**: "어떤 게 나한테 맞을까?" — 40~60대 결정 장벽을 낮추는 호기심 유발형
- **하단 소형 배지**: 붉은 원형 배지에 "돌침대·소파 전용 | 특허·디자인 등록" 표기로 신뢰도 강화
- **인물 인서트 (옵션)**: 오른쪽 하단 코너에 50대 초반 여성이 Cozy-라인에 등 기댄 미소 컷 — 제품 신뢰감 + 타겟 자기동일시
- **썸네일 A/B 후보**: A안 (쿠션 2개 나란히 + 라벨 비교형) vs B안 (여성 모델 + 한쪽 쿠션 강조 단독형) — 클릭률 비교 후 상위 안 채택

## 커뮤니티 성장 전략

**오늘의 제안: "어떤 쿠션이 내 스타일?" 퀴즈형 커뮤니티 게시물 + 라인별 팬덤 형성 전략**

1. **커뮤니티 탭 퀴즈형 게시물 업로드**
   - 질문: "따숨마루 두 가지 중 어떤 게 더 끌리세요?"
   - 선택지 A: "A-라인 — 깔끔하고 단정한 게 좋아요" / B: "Cozy-라인 — 포근하게 감싸주는 게 좋아요"
   - 목적: 라인별 선호도 데이터 수집 + 구매 결정 유도 퍼널 강화 + 댓글 참여로 알고리즘 가중치 확보

2. **댓글 상호작용 시나리오**
   - "어떤 걸 사야 할지 모르겠어요" 댓글 → 체형·생활환경 2가지 질문으로 맞춤 추천 제공 ("돌침대에서 주로 쓰신다면 A-라인을, 소파나 침대 모두 쓰신다면 Cozy-라인이 잘 맞으실 거예요!")
   - 소파 사용 관련 질문 댓글 → 오늘 쇼츠 영상 링크와 함께 소파 세팅법 간단 설명
   - 두 라인 모두 구매한 고객 댓글 → "두 가지 경험 나눠주셔서 감사해요! 혹시 영상에서 소감 짧게 나눠주실 수 있을까요?" — UGC(사용자 제작 콘텐츠) 자연스러운 유도

3. **비교 영상 → 구매 전환 퍼널 설계**
   - 영상 설명란 상단에 두 라인 스마트스토어 직접 링크 각각 배치 ("A-라인 바로가기" / "Cozy-라인 바로가기")
   - 영상 내 타임스탬프를 라인별로 나눠 목차 댓글 고정 핀 → 재시청율 및 시청 지속 시간 지표 개선

4. **신규 구독자 전환 전략**
   - 비교 영상은 구매 직전 단계 사용자(BOFU)를 타겟하는 콘텐츠 → 영상 말미에 "구독하면 A/B 라인 실제 사용 후기 영상도 알림 받으실 수 있어요" 자연스러운 구독 유도 삽입
